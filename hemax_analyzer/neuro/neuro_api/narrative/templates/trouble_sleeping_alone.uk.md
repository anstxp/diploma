# {% if sleep_latency_min and sleep_latency_min > 30 %}Утруднене засинання та порушена якість сну{% else %}Підвищений ризик проблем зі сном{% endif %}

Модель оцінила ймовірність хронічних проблем зі сном (trouble sleeping high) як **{{ p_trouble_sleeping_high }}%** ({% if ratio_trouble_sleeping_high %}×{{ ratio_trouble_sleeping_high }} відносно середнього{% else %}підвищений ризик{% endif %}). На відміну від короткого сну (sleep deficiency — мало годин) це про **якість** сну: чи засинаєте без зусиль, чи прокидаєтесь уночі, чи відчуваєте себе відпочившою вранці.

{% if sleep_latency_min %}Ваш час засинання: **{{ sleep_latency_min }} хв**. {% if sleep_latency_min > 30 %}Норма — до 20 хв. Засинання довше за 30 хв — це **інсомнія за визначенням**, якщо це повторюється ≥3 ночі/тиждень понад 3 місяці.{% endif %}{% endif %}
{% if night_awakenings and night_awakenings >= 3 %}Ви прокидаєтесь **{{ night_awakenings }} разів за ніч** — це фрагментує цикл сну й знижує відновну функцію.{% endif %}
{% if early_morning_awakening %} Раннє прокидання без можливості заснути знову — **класична ознака депресивного епізоду**, особливо у поєднанні з пригніченим настроєм вранці.{% endif %}

{% if on_anxiolytics or on_sleep_meds %}Ви вже приймаєте препарати, що впливають на сон. Це **не лікує** хронічну інсомнію, а тільки маскує симптоми, тому довгострокова стратегія — інша.{% else %}Хронічна інсомнія — це не "недосип через стрес". Це окремий медичний стан, який має ефективніше лікування, ніж снодійні: **CBT-I** (когнітивно-поведінкова терапія інсомнії), яка переважно структурована як 6-8 сесій.{% endif %}
---
actions:
- **Стабільний графік** — лягайте і прокидайтесь у той самий час, **навіть на вихідних** (±30 хв). Це найсильніший фактор регуляції сну
{% if sleep_latency_min and sleep_latency_min > 30 %}- **Правило 20 хвилин**: якщо за 20 хв не заснули — встаньте, зробіть щось спокійне при тьмяному світлі, повертайтесь у ліжко тільки коли захочете спати. Ліжко = тільки сон.
- Не намагайтесь "присилувати" сон — це найшвидший шлях до тривожного очікування сну, яке потім стає окремою проблемою{% endif %}
{% if caffeine_servings_per_day and caffeine_servings_per_day >= 2 %}- Ви вживаєте {{ caffeine_servings_per_day }} порц. кофеїну/день — період напіввиведення 5-6 годин, тому **жодних після 14:00**{% endif %}
{% if screen_time_before_bed_min and screen_time_before_bed_min >= 30 %}- {{ screen_time_before_bed_min }} хв екранного часу перед сном — синє світло пригнічує мелатонін, спробуйте знизити за 60 хв до сну{% endif %}
{% if alcohol_regular %}- Алкоголь регулярно — він допомагає заснути, але **руйнує другу половину ночі** (REM-сну і глибокого сну). Спробуйте 2 тижні без — і порівняйте суб'єктивну "свіжість" вранці{% endif %}
- **CBT-I** (когнітивно-поведінкова терапія інсомнії) — більш ефективна за снодійні у довгостроковій перспективі, evidence-base дуже міцна. Шукайте сертифікованого спеціаліста або CBT-I додатки (Sleepio, Stellar Sleep)
- Прохолодна спальня (18-20°C), тиша, темрява — банально, але працює
---
red_flags:
- Сильна **денна сонливість**, мікросни вдень, неможливість сконцентруватися — впливає на безпеку (водіння, робота з технікою)
- **Раннє ранкове прокидання + пригнічений настрій + втрата інтересу** — обстежитися на депресію, не на інсомнію
- Залежність від снодійних, особливо бензодіазепінів — **поступове зниження дози тільки під наглядом лікаря**, різке припинення може дати rebound insomnia і тривогу
- Якщо проблеми зі сном ≥3 ночі/тиждень понад 3 місяці — це **хронічна інсомнія**, лікувати треба як окремий стан
