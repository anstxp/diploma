# Денна дисфункція — втома, сонливість, знижена продуктивність
icon: 😪
severity: medium
---
Модель виявила підвищений ризик денної сонливості та втоми ({{ p_daytime_dysfunction }}%, ×{{ ratio_daytime_dysfunction }}). Це сигнал, що нічний сон **не відновлює** як має.

{% if sleep_hours_avg and sleep_hours_avg >= 7 and sleep_hours_avg <= 9 %}Цікаво: ви повідомили нормальну тривалість сну ({{ sleep_hours_avg }} год), але втома все одно присутня. Це означає, що **проблема в якості сну, а не у його кількості** — типово для апное, циркадної дисрегуляції, депресії або медичних причин (гіпотиреоз, анемія, дефіцит B12).
{% elif sleep_hours_avg and sleep_hours_avg < 6.5 %}Логічно: ви спите {{ sleep_hours_avg }} год — це нижче за норму, тому втома закономірна. Перший крок — додати години сну.
{% endif %}
{% if snoring and witnessed_apnea %}**Хропіння + засвідчені партнером зупинки дихання** = **класичний обструктивний апное сну (OSA)**. Це найчастіша причина денної втоми у дорослих, особливо з ІМТ >27. Терміновий референс до сомнолога й полісомнографія.
{% elif snoring %}Ви хропете — у поєднанні з денною втомою це **сильний натяк на апное сну**, особливо якщо є зайва вага або великий шийний обхват.
{% endif %}
{% if has_hypothyroidism or on_thyroid_meds %}У вашому профілі вказаний гіпотиреоз — **переконайтесь що доза левотироксину адекватна** (контрольний TSH).
{% endif %}
{% if has_diabetes %}При діабеті денна втома може бути ознакою недостатнього глікемічного контролю (HbA1c >7) або гіпоглікемій вночі.
{% endif %}

Серед топ-факторів моделі — **стать** (жінки частіше скаржаться) і **RDW** (маркер хронічного запалення). {% if ferritin and ferritin < 30 %}Ваш феритин {{ ferritin }} нг/мл — низький, дефіцит заліза часто проявляється саме втомою.{% endif %}

actions:
- Перевірити **феритин**, **вітамін D**, **TSH**, **B12** — найпоширеніші скриті причини втоми
{% if snoring %}- **Полісомнографія або home sleep test** — золотий стандарт діагнозу апное{% endif %}
{% if not exercise_minutes_per_week or exercise_minutes_per_week < 90 %}- Помірна аеробна активність {{ "(хоча б 90 хв/тиждень)" if not exercise_minutes_per_week else "(зараз {} хв/тиж — додайте до 150)".format(exercise_minutes_per_week) }} — парадоксально дає більше енергії{% endif %}
- Сонячне світло вранці 10-15 хв — регулює циркадні ритми краще за каву
- Стабільний графік сну, 7-9 годин на регулярній основі
- Уникайте довгих денних снів (>30 хв) — фрагментують нічний сон
{% if caffeine_servings_per_day and caffeine_servings_per_day >= 4 %}- Ви вживаєте {{ caffeine_servings_per_day }} порц. кофеїну — парадоксально це **посилює денну втому** через "кофеїновий цикл". Поступово знижуйте{% endif %}
{% if alcohol_regular %}- Алкоголь регулярно — основна причина "розбитого" пробудження{% endif %}
- Якщо втома хронічна (>6 міс) попри ці заходи — повне обстеження у сімейного лікаря (включно з оцінкою депресії)
---
red_flags:
- Засинання за кермом — **припиніть водіння** до медичної оцінки
{% if snoring %}- Хропіння або зупинки дихання за словами партнера — **до сомнолога**{% endif %}
- Хронічна втома >6 місяців із міалгіями — синдром хронічної втоми / фіброміалгія?
- Втрата ваги без причини, нічна пітливість + втома — потребує невідкладного обстеження
{% if has_diabetes %}- Раптова сильна втома + спрага + часте сечовипускання — можлива декомпенсація діабету{% endif %}
