Модель виявила **одночасний дефіцит сну та денну дисфункцію** — це класичний патерн розладу сну.

{% if snoring and witnessed_apnea %}🚨 **Ключова знахідка:** ви хропете, і ваш партнер бачив **зупинки дихання уві сні**. Це **класична картина обструктивного апное сну (OSA)** — він спричиняє десятки мікропробуджень за ніч, які ви можете не пам'ятати, але які повністю руйнують відновний сон. **OSA значно підвищує ризик гіпертонії, інфаркту, інсульту, ДТП**. Це не "просто хропіть" — це **медичний стан, який лікують**.
{% elif sleep_latency_min and sleep_latency_min > 30 and night_awakenings and night_awakenings >= 3 %}Ваш патерн виглядає як **хронічна інсомнія**:
- Засинання займає {{ sleep_latency_min }} хв (норма ≤20)
- Пробуджень за ніч: {{ night_awakenings }} (норма 0-2)
Це **DSM-5 діагностичні критерії** інсомнії, якщо триває ≥3 ночі/тиждень понад 3 місяці.
{% else %}Це може бути інсомнія, обструктивне апное, фрагментований сон або циркадний розлад — кожен з цих діагнозів потребує специфічного підходу.
{% endif %}

Хронічно поганий сон не лише знижує якість життя — він **напряму підвищує ризик** гіпертонії, діабету 2 типу, депресії, когнітивних порушень, нещасних випадків.

{% if shift_work %}Ваш позмінний графік **значно ускладнює** ситуацію — циркадний ритм у людини не повністю адаптується до зміни графіка, навіть за роки роботи.{% endif %}
{% if recent_life_stressors %}Ви повідомили про значущі життєві стресори — гострий стрес часто запускає інсомнію, яка потім стає хронічною за принципом "вивченої реакції".{% endif %}

actions:
- **Запис до сімейного лікаря або сомнолога**{% if snoring and witnessed_apnea %} — **протягом 2 тижнів, не відкладайте**{% endif %}
{% if snoring %}- **Полісомнографія** (нічне дослідження сну) — золотий стандарт діагнозу. Альтернатива — home sleep test{% endif %}
- Перевірте рівень **феритину**, **вітаміну D**, **TSH**, **B12** — їх дефіцит частий при розладах сну
- Стабільний графік сну (засинати і прокидатися у той самий час, ±30 хв навіть на вихідних)
- 7-9 годин сну на добу для дорослих
- Темна, прохолодна (18-20°C) спальня; без екранів за годину до сну
- Кофеїн — не пізніше 14:00; алкоголь — обмежити (він фрагментує сон)
{% if not snoring %}- Якщо є тривога перед сном — **CBT-I** (когнітивно-поведінкова терапія інсомнії) має кращу evidence-base за снодійні. Шукайте онлайн-програми CBT-I (наприклад Sleepio){% endif %}
{% if on_sleep_meds or uses_sleep_meds %}- Поточні снодійні: обговоріть з лікарем перегляд — **бензодіазепіни** не рекомендовані як перша лінія, особливо у літніх (ризик падінь, делірію){% endif %}
