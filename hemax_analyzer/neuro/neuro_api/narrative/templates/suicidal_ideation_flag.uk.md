# ⚠ Сигнали, що потребують підтримки фахівця
icon: 🆘
severity: high
---
Модель виявила **показники, що корелюють із думками про самопошкодження або суїцид**.

Це **не діагноз** — модель оцінює лише ймовірність на основі лабораторних і антропометричних даних і **не замінює клінічну оцінку**. Однак сам цей сигнал — достатня причина поговорити з близькою людиною або фахівцем.

{% if recent_life_stressors %}Ви повідомили про значущі життєві стресори за останні 6 місяців — це **підвищує гостроту** цього сигналу. Кризові періоди тимчасові, навіть коли здаються нескінченними.
{% endif %}
{% if on_antidepressants %}Ви приймаєте антидепресанти. **Важливо знати:** у перші 1-2 тижні початку терапії (особливо у молодих людей до 25) суїцидальні думки можуть **тимчасово посилюватись** — це відомий побічний ефект, який зазвичай минає. **Не припиняйте препарати раптово**, але **обов'язково повідомте лікаря** про ці думки сьогодні-завтра.
{% endif %}
{% if social_support_low %}Ви повідомили про брак близьких людей у вашому житті. Прямо зараз це здається безвихіддю, але **дзвінок на кризову лінію (7333) — це теж "близька людина" у цей момент**. Не треба знати когось, щоб говорити.
{% endif %}
{% if has_chronic_pain or chronic_pain %}Хронічний біль значно підвищує ризик суїцидальних думок — це не слабкість, а **закономірна нейробіологічна реакція**. Лікування болю покращує і психічний стан.
{% endif %}

**Якщо ви зараз в кризі або маєте думки про самопошкодження — будь ласка, зверніться по допомогу негайно. Ви не самі.**
---
actions:
- 📞 **Lifeline Україна — 7333** (безкоштовно з мобільного, цілодобово, конфіденційно)
- 📞 **Гаряча лінія Lifeline Ukraine — 0 800 200 500**
- 📞 **Експрес-допомога для військових і ветеранів — 116 123**
- 🚑 **Швидка допомога — 103** (або **112**) у разі загрози життю
- Зверніться до сімейного лікаря, психіатра або психотерапевта **протягом 1-2 днів**
{% if on_antidepressants %}- **Зв'яжіться зі своїм лікарем сьогодні** — повідомте про думки. Це **не означає що препарат поганий**, але можливо потрібно скоригувати дозу або додати інший препарат
{% endif %}
- Розкажіть близькій людині — другу, родичу — про те, як ви себе почуваєте
- Не залишайтеся самі тривалий час; домовтесь побути з кимось у найближчі дні
- Уникайте алкоголю та психоактивних речовин — вони **підсилюють імпульсивність** і значно підвищують ризик дії
- **Приберіть з легкого доступу** предмети, що можуть бути використані для самопошкодження. Якщо ви чи близька людина — попросіть когось забрати їх з дому тимчасово
- Створіть **safety plan**: список 3 людей, яким можна подзвонити; список 3 речей, що допомагають заспокоїтись; план що робити в моменти кризи
---
red_flags:
- **Конкретний план**, як завдати собі шкоди — **викличте 103 / 112 негайно**
- Раптове відчуття **"полегшення" або "спокою"** після важкого періоду — це може означати рішення про дію, ризиковий знак
- Соціальна ізоляція, **прощання з близькими, роздавання речей**, написання заповіту без об'єктивної потреби
- Безсоння та виснажливі думки кілька днів поспіль
- **Доступ до зброї, медикаментів у великих кількостях** — попросіть близьку людину зберігати їх
{% if on_antidepressants %}- Раптова сильна тривога, агітація, безсоння у перші 1-2 тижні антидепресанту — **до лікаря сьогодні**
{% endif %}
