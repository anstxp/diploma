# Критично знижені нейтрофіли — підвищений ризик інфекцій
icon: 🔴
severity: high
---
Рівень нейтрофілів ({{ anc }} ×10⁹/л) суттєво знижений. Нейтрофіли — головний захист організму від бактеріальних і грибкових інфекцій. При таких низьких значеннях інфекції можуть розвиватись **швидко і важко**, навіть із незначних причин.

{% if takes_chemotherapy %}
**Це очікуваний ефект хіміотерапії** — препарати пригнічують кістковий мозок. Але **очікуваність не означає безпеку**: ризик фебрильної нейтропенії (інфекція + гарячка на тлі знижених нейтрофілів) залишається загрозливим для життя.

Будь-яка температура ≥38°C на цій фазі = **виклик швидкої або негайно у приймальне**, без винятків. Не приймайте парацетамол щоб «збити температуру і подивитись» — це маскує сигнал. Більшість онкоцентрів мають прямий телефон для таких ситуацій.
{% elif on_corticosteroids %}
Стероїди зазвичай **підвищують** нейтрофіли, не знижують. Тому таке зниження на тлі стероїдів — нетипове, потребує перегляду інших причин (вірусна інфекція, поєднання з іншими ліками).
{% elif has_autoimmune %}
У вас аутоімунне захворювання — нейтропенія може бути і самим проявом хвороби (особливо при системному червоному вовчаку), і побічним ефектом терапії (метотрексат, азатіоприн). Потрібна оцінка ревматолога.
{% else %}
Причини бувають різними: побічна дія ліків (особливо хіміотерапії, деяких антибіотиків і нейролептиків), вірусні інфекції, автоімунні захворювання, захворювання кісткового мозку.
{% endif %}
---
actions:
{% if takes_chemotherapy %}
- **При будь-якій температурі ≥38°C — НЕГАЙНО у приймальне відділення** (фебрильна нейтропенія — невідкладний стан)
- Зв'яжіться з вашим онкологом сьогодні щоб обговорити поточні значення
- Уникайте натовпів, людей з симптомами застуди, сирої їжі (овочі/риба/м'ясо)
- Ретельна гігієна: миття рук, обробка рота, догляд за нігтями
- Обговоріть з лікарем G-CSF (філграстим) — препарат що стимулює нейтрофіли
- Тримайте номер екстреного зв'язку онкоцентру під рукою
{% elif on_corticosteroids %}
- **Зверніться до лікаря сьогодні** — таке зниження на стероїдах нетипове
- Здайте: вірусні маркери (ЦМВ, ВЕБ), повторно ЗАК
- Не припиняйте стероїди самостійно — це окрема небезпека
{% elif has_autoimmune %}
- **Зверніться до ревматолога** з результатами
- Перегляньте всі ваші препарати — деякі можуть викликати нейтропенію
- Уникайте інфекційних контактів до перегляду терапії
{% else %}
- **Негайно** звʼяжіться з вашим лікарем (сьогодні)
- Якщо ви нещодавно починали нові ліки — повідомте про це
- До консультації: уникайте скупчень людей, не готуйте сиру їжу для інших, ретельно мийте руки
- Звичайних профілактичних заходів уже замало — ідіть до лікаря
{% endif %}
---
red_flags:
- Температура понад 38°C — **негайно** у приймальне відділення, не чекайте
- Озноб, пітливість, відчуття «важкої» хвороби
- Біль або почервоніння у будь-якому місці — можлива локальна інфекція
{% if takes_chemotherapy %}
- Біль у горлі, утруднене ковтання, виразки рота — мукозит з накладеною інфекцією
- Будь-яка кровотеча — на тлі хіміотерапії часто супутня тромбоцитопенія
{% endif %}
