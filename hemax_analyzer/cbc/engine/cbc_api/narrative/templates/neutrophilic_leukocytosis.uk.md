# Нейтрофільний лейкоцитоз — ознака активного запалення
icon: 🔥
severity: medium
---
Ваші **лейкоцити підвищені переважно за рахунок нейтрофілів** — це класична картина **нейтрофільного зсуву**. Найчастіше це **реакція організму на запалення або інфекцію**, рідше — стрес-реакція чи побічна дія ліків.

**Поширені «побутові» джерела** (з них зазвичай і починають пошук):
- **Дихальні шляхи** — застуда, бронхіт, синусит, пневмонія
- **ЛОР** — отит, тонзиліт, фарингіт
- **Зуби/ясна** — пародонтит, абсцес кореня, стоматит
- **Шкіра** — фурункули, інфіковані рани, цілюліт
- **Сечові шляхи** — цистит, пієлонефрит
- **ШКТ** — гастроентерит, харчове отруєння

**Рідше і важче** (потребують оцінки лікаря):
- Гострий апендицит, холецистит, дивертикуліт
- Абсцеси внутрішніх органів, септичні вогнища
- Глибокі інфекції м'яких тканин

**Не-інфекційні причини:**
- Стрес-реакція — після травми, операції, сильного болю, інтенсивних навантажень
- Кортикостероїди, літій, G-CSF — фармакологічно піднімають нейтрофіли

{% if esr_high or crp_high %}
**Додатково:** {% if esr_high %}ШОЕ також підвищена{% endif %}{% if esr_high and crp_high %}, а також {% endif %}{% if crp_high %}СРБ підвищений{% endif %} — це **підтверджує** запальний контекст і підсилює потребу у пошуку джерела.
{% endif %}

Це **не** «дрібне відхилення». Поєднання підвищених WBC + переважання нейтрофілів — клінічно значимий сигнал, який потребує оцінки лікаря.
---
actions:
- **Подумайте про симптоми останніх 1-2 тижнів**: температура, біль (де?), кашель, виділення, дизурія, рани, проблеми зі стільцем
- Здайте **С-реактивний білок (СРБ)** — швидко зростає при бактеріальних інфекціях; разом з нейтрофільним лейкоцитозом це сильніший доказ активної інфекції
- Якщо є локальні симптоми — додатково за призначенням: загальний аналіз сечі, посів сечі/мокроти, УЗД відповідної ділянки
- **Зверніться до сімейного лікаря або терапевта найближчим часом** — для пошуку джерела
- При гарячці ≥38°C, погіршенні самопочуття — **сьогодні** до лікаря або у приймальне
- Не приймайте антибіотики самостійно — для правильного вибору потрібна етіологія
- Якщо приймаєте кортикостероїди, нещодавно перенесли операцію/травму, або були у сильному стресі — це може давати реактивне підвищення без інфекції; обговоріть з лікарем
- Повторіть CBC через 5-7 днів — для оцінки динаміки
---
red_flags:
- Температура ≥38°C, особливо з ознобом, пітливістю
- Сильний локальний біль (живіт, поперек, груди, голова) — **сьогодні** до лікаря
- Сплутаність свідомості, сонливість, прискорене дихання — можливий сепсис, **виклик швидкої**
- Бліда, мармурова шкіра, холодні кінцівки на тлі гарячки — терміново до лікаря
- Сильне нездужання, неможливість стояти — **EMERGENCY**
