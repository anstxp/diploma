# Ознаки дефіциту заліза
icon: 🩸
severity: medium
---
Ваш гемоглобін — {{ hgb }} г/дл — {% if hgb_low %}нижчий за нижню межу норми ({{ hgb_low }}){% else %}знижений{% endif %}. При цьому ваші еритроцити дрібніші та містять менше гемоглобіну, ніж очікується — це класична картина **дефіциту заліза**.

{% if pregnant %}
Під час вагітності зростає потреба в залізі — для плода і збільшення об'єму крові. Дефіцит заліза у вагітних дуже поширений, але **не варто його ігнорувати** — він підвищує ризик передчасних пологів і впливає на розвиток дитини. Препарати заліза для вагітних безпечні і часто потрібні.
{% elif on_oral_anticoagulants %}
Ви приймаєте антикоагулянти, тому **дефіцит заліза при цьому особливо насторожує** — потрібно активно шукати приховані крововтрати. Пацієнти на варфарині/DOACах мають підвищений ризик прихованих кровотеч у ШКТ.
{% elif has_ckd %}
У вас хронічна хвороба нирок — саме нирки виробляють еритропоетин (гормон, який стимулює кровотворення). При ХХН часто розвивається анемія, яка не завжди є дефіцитом заліза. Потрібен повний залізний профіль (феритин + насичення трансферину) для розрізнення.
{% elif is_female and age and age < 50 %}
У жінок репродуктивного віку найчастіша причина — **крововтрати з рясних або тривалих місячних**. Друга за поширеністю — недостатнє надходження заліза з їжею (особливо при вегетаріанських дієтах).
{% elif is_male or (age and age >= 50) %}
У чоловіків та жінок старше 50 років дефіцит заліза **завжди потребує з'ясування причини** — найчастіше це приховані крововтрати зі шлунково-кишкового тракту (виразки, поліпи, рідше — пухлини). Цю причину треба активно шукати.
{% elif age and age < 18 %}
У вашому віці дефіцит заліза часто пов'язаний зі швидким ростом і недостатнім надходженням заліза з їжею. Втім, інші причини теж треба виключити.
{% endif %}

Самостійно з'ясувати конкретну причину за цим аналізом неможливо — потрібні додаткові дослідження.

actions:
- Здайте додаткові аналізи: **феритин** (найінформативніший), залізо сироватки, трансферин, ЗЗЗС
{% if has_ckd %}
- Здайте також креатинін, ШКФ, добову протеїнурію — щоб оцінити стадію ХХН і потребу в еритропоетині
- Зверніться до нефролога — анемія при ХХН може потребувати спеціального лікування
{% elif on_oral_anticoagulants %}
- Терміново обговоріть з лікарем — можуть знадобитися гастроскопія/колоноскопія для пошуку джерела
- Не змінюйте дозу антикоагулянтів самостійно
{% elif pregnant %}
- Обговоріть з акушером-гінекологом призначення препаратів заліза у безпечному дозуванні
- Контроль гемоглобіну кожні 4-6 тижнів
{% else %}
- Зверніться до терапевта з результатами цих аналізів
- {% if is_male or (age and age >= 50) %}Обговоріть з лікарем необхідність гастроскопії / колоноскопії для пошуку джерела крововтрати{% else %}Якщо місячні рясні — обговоріть це з гінекологом{% endif %}
{% endif %}
- Не починайте прийом заліза самостійно — надлишок заліза теж шкідливий
- До початку лікування: їжте більше продуктів, багатих на залізо (червоне м'ясо, печінка, бобові, листова зелень); вживайте їх із продуктами, багатими на вітамін С (цитрусові, болгарський перець); уникайте чаю та кави під час їжі (вони погіршують всмоктування заліза)
