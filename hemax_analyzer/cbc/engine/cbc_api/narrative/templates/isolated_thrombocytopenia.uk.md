# Тромбоцити знижені
icon: 🟡
severity: low
---
Рівень тромбоцитів ({{ plt }} ×10⁹/л) нижче норми, але не критично низький.

{% if has_liver_disease %}
**У вашому профілі вказана хвороба печінки — це найімовірніша причина.** При цирозі та хронічному гепатиті селезінка збільшується (спленомегалія) і **накопичує тромбоцити** в собі. Плюс печінка виробляє менше тромбопоетину — гормону, що стимулює утворення тромбоцитів.

Рівень тромбоцитів **корелює зі стадією фіброзу** — це маркер прогресування печінкової хвороби, який варто відстежувати разом з лікарем.
{% elif on_oral_anticoagulants %}
Ви приймаєте антикоагулянти — поєднання навіть **помірно знижених тромбоцитів** з антикоагулянтною терапією підвищує ризик кровотеч. Це не emergency на цих цифрах, але потрібна консультація лікаря для оцінки балансу ризику тромбозу і кровотечі.
{% elif has_autoimmune %}
У вас аутоімунне захворювання — імунна тромбоцитопенія (ІТП) може бути проявом основної хвороби або побічним ефектом терапії (метотрексат, азатіоприн, гідроксихлорохін).
{% else %}
Можливі причини: нещодавня вірусна інфекція, дефіцит фолатів/B12, побічна дія ліків, імунна тромбоцитопенія (ІТП), захворювання печінки, іноді — індивідуальна особливість.
{% endif %}

actions:
{% if has_liver_disease %}
- **Покажіть результати гепатологу** — оцінити стадію печінкової хвороби
- Розгляньте з лікарем **FibroScan** або еластометрію — для оцінки фіброзу
- УЗД печінки і селезінки — оцінити розмір селезінки (норма <12 см)
- Здайте: АЛТ, АСТ, білірубін, альбумін, INR — повна оцінка функції печінки
- Уникайте аспірину, НПЗП, алкоголю — все це посилює ризик кровотеч
- Стоматологічні втручання — попередьте лікаря про знижені тромбоцити
{% elif on_oral_anticoagulants %}
- Зверніться до лікаря який призначив антикоагулянт — обговоріть моніторинг
- Не припиняйте препарат самостійно
- Уникайте інших препаратів що знижують згортання (аспірин, НПЗП)
- Повторіть аналіз через 1-2 тижні
{% else %}
- Повторіть аналіз через 1-2 тижні, переважно у тій самій лабораторії
- Обмежте прийом **аспірину, ібупрофену**, якщо можете
- Якщо повторний аналіз також низький — зверніться до терапевта
- Перевірте функцію печінки і рівень B12/фолатів
{% endif %}

red_flags:
- Дрібні червоні точки на шкірі (петехії)
- Часті синці, кровоточивість ясен або носові кровотечі
- Рясні й тривалі місячні
{% if has_liver_disease %}
- Кровотеча з ясен/носа що довго не зупиняється — оцінити коагулограму
- Поява асциту, набряків ніг, жовтушність — декомпенсація печінки, **до лікаря**
- Чорний кал, блювання з кров'ю — можлива кровотеча з варикозу стравоходу, **EMERGENCY**
{% endif %}
