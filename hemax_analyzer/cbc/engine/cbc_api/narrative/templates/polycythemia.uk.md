# Гемоглобін і еритроцити вище норми
icon: 🔺
severity: medium
---
Ваш гемоглобін — {{ hgb }} г/дл — **вище за верхню межу норми**. Такий стан називається еритроцитозом.

{% if smoker %}
**Найімовірніша причина — куріння.** Хронічне куріння призводить до помірної гіпоксії (CO зв'язується з гемоглобіном замість кисню), і кістковий мозок реагує на це **посиленою продукцією еритроцитів**. Це **компенсаторна реакція**, не справжня хвороба, але вона додає серцево-судинного ризику.

Кинути курити — найефективніший спосіб **знизити еритроцитоз**. Зазвичай через 3-6 місяців після відмови показники нормалізуються самостійно.
{% elif has_cardiovascular_disease %}
У вас є серцево-судинна хвороба — підвищені еритроцити **збільшують ризик тромбозів** і знижують ефективність терапії. Потрібен прицільний пошук причини: апное сну, ХОЗЛ, інші стани з гіпоксією.
{% else %}
Можливі причини:
- **Вторинні**: хронічна нестача кисню (куріння, апное сну, захворювання легень, проживання на великій висоті)
- **Відносні**: зневоднення (часто оманливо підвищує показники)
- **Первинні**: істинна поліцитемія — рідкісне захворювання кісткового мозку, частіше у людей після 50
{% endif %}

{% if signals.thrombocytosis or signals.leukocytosis %}
У вас також підвищені інші клітини крові — це підсилює підозру на справжню поліцитемію. Важливо зʼясувати це з лікарем, бо лікування істотно відрізняється від вторинних причин.
{% endif %}

actions:
{% if smoker %}
- **Розгляньте відмову від куріння** — це найвпливовіший фактор, обговоріть з лікарем нікотинзамісну терапію
- Здайте сатурацію O₂ і пульсоксиметрію — щоб виключити інші причини гіпоксії
- Здайте феритин — еритроцитоз курців часто має нормальний JAK2
- Повторіть аналіз через 3 місяці після відмови
- При наявності храпу/денної сонливості — обстеження на апное сну
{% elif has_cardiovascular_disease %}
- **Терміново до кардіолога** — оцінити ризик тромбозу, можливо корекція терапії
- Обстеження на апное сну (часта причина і у ССЗ)
- Здайте JAK2 — для виключення істинної поліцитемії
- Перевірте сатурацію O₂ під час сну
{% else %}
- Зверніться до терапевта — йому потрібен повний клінічний огляд
- Якщо курите — скажіть лікарю, як часто й як давно
- Обговоріть якість сну (можлива причина — апное)
- Можливо знадобиться консультація гематолога і аналіз на мутацію **JAK2**
{% endif %}
