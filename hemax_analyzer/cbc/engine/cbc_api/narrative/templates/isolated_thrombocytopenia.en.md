# Low platelet count
icon: 🟡
severity: low
---
Your platelet count ({{ plt }} ×10⁹/L) is below normal, but not critically low. Possible causes: recent viral infection, folate/B12 deficiency, medication side effect, immune thrombocytopenia (ITP), liver disease, sometimes individual variation.
---
actions:
- Repeat the test in 1-2 weeks, ideally at the same lab
- Limit **aspirin, ibuprofen** use if possible
- If still low on repeat, see a GP
- Check liver function and B12/folate
---
red_flags:
- Small red dots on skin (petechiae)
- Frequent bruising, bleeding gums, or nosebleeds
- Heavy and prolonged menstrual periods
