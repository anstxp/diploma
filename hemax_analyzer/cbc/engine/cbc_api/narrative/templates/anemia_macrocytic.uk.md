# Анемія з великими еритроцитами — ймовірно брак B12 або фолатів
icon: 🩸
severity: medium
---
Ваш гемоглобін знижений, а еритроцити **більші за норму** (MCV = {{ mcv }} fL). Така картина найчастіше виникає при браку **вітаміну B12** або **фолієвої кислоти** — обидва потрібні для нормального дозрівання еритроцитів у кістковому мозку.

{% if has_diabetes and on_oral_diabetics %}
Ви приймаєте оральні препарати від діабету — найчастіше це **метформін**. **Тривалий прийом метформіну (5+ років) знижує всмоктування B12 у ~30% пацієнтів** через вплив на тонкий кишечник. Це найімовірніша причина у вашому випадку. Хороша новина: дефіцит B12 легко лікується ін'єкціями або високими дозами препаратів.
{% elif age and age >= 65 %}
У вашому віці дефіцит B12 зустрічається частіше — близько 10-15% людей після 65 мають знижене всмоктування B12 у шлунку (атрофічний гастрит). Часто потребує довготривалого приймання B12.
{% elif on_oral_anticoagulants %}
Ви приймаєте антикоагулянти — на тлі цього **навіть незначна кровотеча небезпечна**, хоча зазвичай це не основна причина макроцитарної анемії. Все ж потрібно виключити приховані крововтрати.
{% else %}
Інші можливі причини: регулярне вживання алкоголю, деякі лікарські препарати (метформін при тривалому прийомі, метотрексат), захворювання щитоподібної залози. Рідше — більш серйозні захворювання кісткового мозку.
{% endif %}
---
actions:
- Здайте: **вітамін B12**, **фолати** (в сироватці), ТТГ (щитовидка), і за можливості — гомоцистеїн
{% if has_diabetes and on_oral_diabetics %}
- Скажіть ендокринологу про дефіцит — у пацієнтів на метформіні часто рекомендують щорічну перевірку B12
- Якщо B12 справді знижений — лікування ін'єкціями (1000 мкг/міс) або великими дозами орально
- Не припиняйте метформін самостійно — лікування дефіциту B12 не вимагає його відміни
{% else %}
- Зверніться до терапевта з цими результатами
- Якщо ви вживаєте алкоголь регулярно — це важлива інформація для лікаря
- Складіть список усіх препаратів (включно з БАДами), які приймаєте
{% endif %}
