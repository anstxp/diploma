# Анемія без змін розміру еритроцитів
icon: 🩸
severity: medium
---
Ваш гемоглобін знижений, але розмір еритроцитів залишається нормальним (MCV = {{ mcv }} fL). Така картина — нормоцитарна анемія — сама по собі не вказує на конкретну причину, але часто зустрічається при:

- **Хронічних захворюваннях** (запальні процеси, хронічна хвороба нирок, захворювання печінки)
- **Початковій стадії** залізодефіциту (до того як еритроцити встигли зменшитись)
- **Анемії після гострої крововтрати**
- **Гемолізі** (прискореному руйнуванні еритроцитів)

{% if age and age >= 60 %}
У людей старше 60 нормоцитарна анемія часто повʼязана з хронічною хворобою нирок або запальними процесами.
{% endif %}
---
actions:
- Здайте: **феритин**, залізо, трансферин; **креатинін + швидкість клубочкової фільтрації (ШКФ)** для оцінки нирок; **С-реактивний білок** для оцінки запалення; **ретикулоцити** для оцінки регенерації
- Зверніться до терапевта — йому потрібен буде повний огляд і збір анамнезу, щоб звузити пошук
- Візьміть із собою список всіх хронічних захворювань та препаратів
---
red_flags:
- Значна задишка у спокої або біль у грудях
- Темна сеча або жовтушність шкіри (може вказувати на гемоліз)
- Набряки ніг, задишка при навантаженні (може вказувати на проблеми з нирками)
