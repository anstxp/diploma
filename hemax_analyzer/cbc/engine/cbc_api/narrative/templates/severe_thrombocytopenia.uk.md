# Критично знижені тромбоцити — підвищений ризик кровотеч
icon: 🔴
severity: high
---
Рівень тромбоцитів ({{ plt }} ×10⁹/л) дуже низький. Тромбоцити відповідають за зсідання крові, і при таких значеннях підвищений ризик **спонтанних кровотеч** — з ясен, носа, після дрібних травм.

{% if on_oral_anticoagulants %}
**УВАГА — ВИ ПРИЙМАЄТЕ АНТИКОАГУЛЯНТИ.** Поєднання низьких тромбоцитів з антикоагулянтною терапією **різко підвищує ризик небезпечних кровотеч** — внутрішньочерепних, шлунково-кишкових, у суглоби. Це **невідкладний стан**, потрібна термінова консультація.
{% elif on_corticosteroids %}
Ви приймаєте кортикостероїди — вони можуть впливати на тромбоцити по-різному. Часом це парадоксально знижує їх кількість при тривалому прийомі високих доз. Лікар має оцінити чи це пов'язано з терапією.
{% elif has_liver_disease %}
У вас є хвороба печінки — це **дуже часта причина тромбоцитопенії**. Печінка виробляє тромбопоетин (стимулятор тромбоцитів), а селезінка при цирозі починає накопичувати тромбоцити. Поточні значення критично низькі — потрібна оцінка стану печінки.
{% else %}
Причини: побічна дія ліків, вірусні інфекції, імунна тромбоцитопенія (ІТП), захворювання печінки, селезінки або кісткового мозку.
{% endif %}
---
actions:
{% if on_oral_anticoagulants %}
- **ТЕРМІНОВО — у приймальне відділення сьогодні.** Не чекайте до завтра.
- **НЕ ПРИПИНЯЙТЕ антикоагулянт самостійно** — це може спричинити тромбоз. Лікар прийме рішення.
- Уникайте будь-яких травм, не приймайте інших препаратів без узгодження
- Принесіть список усіх ліків і дозувань
- Якщо є будь-яка кровотеча — це критично, скажіть одразу
{% else %}
- **Зверніться до лікаря сьогодні** — потрібна консультація терапевта і, ймовірно, гематолога
- Уникайте травм, контактних видів спорту, падінь
- Не приймайте **аспірин, ібупрофен** і подібні препарати — вони знижують зсідання
- Повідомте лікаря про всі препарати, які приймаєте
{% endif %}
---
red_flags:
- Дрібні червоні точки на шкірі (петехії) або великі синці без причини
- Кровоточивість ясен, носові кровотечі, довге кровотеча з ранок
- Кров у сечі, стільці або блювотинні — **негайно** в приймальне
- Сильний головний біль, раптова слабкість одної сторони тіла — негайно
{% if on_oral_anticoagulants %}
- Будь-яка нова кровотеча на тлі антикоагулянтів — **терміново у приймальне**
- Сильний біль у животі, спині — можлива внутрішня кровотеча
{% endif %}
