# Картина активної інфекції
icon: 🦠
severity: medium
---
Ваш рівень лейкоцитів підвищений ({{ wbc }} ×10⁹/л), причому переважно за рахунок **нейтрофілів** — це тип білих кров'яних клітин, який першим реагує на бактеріальні інфекції. Така комбінація найчастіше свідчить про:

- **Гостру бактеріальну інфекцію** (найпоширеніша причина)
- Менш ймовірно — гостре запалення без інфекції (травма, інфаркт, операція)
- Рідко — реакція на стрес чи фізичне навантаження

{% if signals.nlr_high %}
Ваш **індекс NLR** (співвідношення нейтрофілів до лімфоцитів) також підвищений — це додатковий маркер активного запалення, який допомагає відрізнити бактеріальну інфекцію від вірусної.
{% endif %}
---
actions:
- Якщо у вас є симптоми (температура, біль, слабкість, кашель тощо) — **зверніться до терапевта сьогодні або завтра**
- Якщо симптомів немає — все одно покажіть результат лікарю найближчим часом, щоб зʼясувати причину
- Пийте достатньо рідини та відпочивайте
- Не приймайте антибіотики без призначення лікаря — не кожна інфекція потребує їх
---
red_flags:
- Висока температура (понад 38.5°C), що не знижується жарознижувальними
- Сильний головний біль + ригідність шиї
- Важка задишка або біль у грудях
- Сплутаність свідомості, сильна слабкість
- Прогресивне погіршення стану
