# Тромбоцитопенія + антикоагулянти = підвищений ризик кровотеч
icon: 🚨
severity: high
---
**Терміново зверніться до лікаря.** Поєднання двох факторів значно підвищує ризик кровотеч:

1. **Знижені тромбоцити** ({% if plt %}PLT {{ plt }} ×10⁹/л{% else %}PLT < 150{% endif %}) — клітини відповідальні за згортання крові
2. **Антикоагулянти** (варфарин, апіксабан, ривароксабан, дабігатран, едоксабан) — спеціально гальмують згортання

{% if thrombocytopenia_sev == "high" %}
Тромбоцити **дуже низькі** — на тлі антикоагулянтної терапії це створює **різко підвищений ризик небезпечних кровотеч** (внутрішньочерепних, шлунково-кишкових, у суглоби). Це **невідкладний стан**.
{% else %}
Поєднання навіть **помірно знижених тромбоцитів** з антикоагулянтною терапією — клінічно значимий сигнал, що потребує оцінки лікаря для перегляду балансу ризику тромбозу і кровотечі.
{% endif %}
---
actions:
{% if thrombocytopenia_sev == "high" %}
- **ТЕРМІНОВО — у приймальне відділення сьогодні.** Не чекайте до завтра.
- **НЕ ПРИПИНЯЙТЕ антикоагулянт самостійно** — це може спричинити тромбоз. Лікар прийме рішення про дозу/відміну/заміну.
- Уникайте будь-яких травм, не приймайте інших препаратів без узгодження
- Принесіть список усіх ліків і дозувань
- Якщо вже є будь-яка кровотеча — це критично, скажіть лікарю одразу
{% else %}
- Зв'яжіться з лікарем сьогодні — потрібна консультація терапевта/гематолога
- **Не припиняйте антикоагулянт самостійно** — це може спричинити тромбоз
- Не приймайте **аспірин, ібупрофен** і подібні препарати — додатково знижують згортання
- Уникайте травм, контактних видів спорту
- Повторіть аналіз через 1-2 тижні разом з гематологом
{% endif %}

red_flags:
- Дрібні червоні точки на шкірі (петехії), великі синці без причини
- Кровоточивість ясен, носові кровотечі, довге кровотечення з ранок
- Кров у сечі, стільці, блювотинні — **негайно** у приймальне
- Чорний дьогтеподібний кал — можлива кровотеча з ШКТ
- Сильний головний біль, раптова слабкість одної сторони тіла — **EMERGENCY**
- Сильний біль у животі, спині — можлива внутрішня кровотеча
- Будь-яка нова кровотеча на тлі антикоагулянтів — **терміново у приймальне**
