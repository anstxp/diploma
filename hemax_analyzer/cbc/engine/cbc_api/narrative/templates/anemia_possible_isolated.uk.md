# Гемоглобін трохи нижче норми
icon: 🩸
severity: low
---
Ваш гемоглобін ({{ hgb }} г/дл) нижче нижньої межі для вашої статі ({{ hgb_low }}), але розмір еритроцитів нормальний і немає явної картини залізодефіциту чи інших типів анемії.

Це може бути ранньою стадією якогось типу анемії, відносним показником (після нещодавнього пиття великої кількості води), або просто індивідуальна нижча норма.
