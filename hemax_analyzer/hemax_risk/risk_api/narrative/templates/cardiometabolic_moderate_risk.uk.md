{% if has_any_chronic %}
У вашому профілі вже є хронічне захворювання, і поточна модельна оцінка показує **помірний ризик** за кількома напрямками. Це означає, що **контроль стабільний**{% if on_any_therapy %}, але є простір для оптимізації терапії{% endif %}.

Це **не означає що ризик зник** — продовжуйте регулярний моніторинг і не припиняйте поточні препарати самостійно.
{% else %}
У вашому профілі **кілька систем з помірно підвищеним ризиком**. Жодне відхилення не катастрофічне, але разом вони формують характерний фенотип, який заслуговує на увагу.

Це **зона тих, у кого ще все можна повернути** — без ліків, лише зміною способу життя. Без корекції факторів ризику ймовірність несприятливого профілю з часом може зростати.
{% endif %}
