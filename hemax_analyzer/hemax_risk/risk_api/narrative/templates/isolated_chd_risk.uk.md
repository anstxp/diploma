icon: 🫀
severity: high
---
{% if has_cardiovascular_disease %}
У вашому профілі вже є серцево-судинне захворювання, і поточна модельна оцінка показує **підвищений ризик ІХС** ({{ p_chd }}%, ×{{ ratio_chd }}). Це означає, що потрібен **позаплановий перегляд вторинної профілактики** — мета цілі жорсткіша, ніж для первинної профілактики.

{% if on_statins or on_acei_arb or on_antihypertensives %}Ви на профілактичній терапії — **не припиняйте препарати самостійно**. Обговоріть з кардіологом потенційне посилення.{% endif %}
{% else %}
Виявлено **підвищений ризик ішемічної хвороби серця (ІХС)** ({{ p_chd }}%, ×{{ ratio_chd }} від популяційного). ІХС — це звуження артерій серця через атеросклероз.

Ризик не означає поточну хворобу — це сигнал поглибленої оцінки.
{% endif %}

actions:
{% if has_cardiovascular_disease %}
- **Кардіолог — позаплановий візит** для перегляду вторинної профілактики
- Якщо давно не робили — ЕКГ, ехокардіографія, можливо CT-коронарографія
- Цільові показники: LDL <55 мг/дл, АТ <130/80
- Антитромбоцитарна терапія (аспірин/клопідогрель) — за призначенням
- **Не пропускайте поточні препарати** — це найчастіша причина повторних подій
{% else %}
- **Кардіолог протягом 2-4 тижнів**
- ЕКГ у спокої, можливо — стрес-тест або CT-коронарографія (CAC score)
- Повний lipid panel + Lp(a)
- Контроль артеріального тиску
- Розгляньте з лікарем профілактичні статини і аспірин
- Спорт, дієта, відмова від куріння — обов'язково
{% endif %}

red_flags:
- Біль/тиск у грудях при навантаженні або у спокої — **103/112 при ірадіації в руку/щелепу**
- Біль у щелепі, лівому плечі
- Задишка при навантаженнях, що раніше не викликали її
- Раптове запаморочення
