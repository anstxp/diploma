{% if has_diabetes %}
У вашому профілі вже вказаний цукровий діабет, поточний модельний ризик помірний ({{ p_diabetes }}%). Це **позитивно** — означає, що ваш контроль глюкози досить стабільний{% if takes_metformin or on_oral_diabetics or on_insulin %}, і цукрознижувальна терапія працює{% endif %}.

Однак це не «зникнення» діабету — продовжуйте моніторинг, прийом препаратів і регулярні візити до ендокринолога.
{% else %}
Ваш профіль вказує на **помірний ризик розвитку діабету** ({{ p_diabetes }}%, ×{{ ratio_diabetes }}). Це **переддіабетична зона** — ще не хвороба, але вже не повноцінна норма.

Це сигнал почати діяти **зараз**, а не чекати погіршення. Більшість людей з цим профілем можуть **повернутися до повної норми** за 6-12 місяців простими змінами способу життя.
{% endif %}

actions:
{% if has_diabetes %}
- Продовжуйте регулярні візити до ендокринолога — раз на 3-6 місяців
- HbA1c раз на 3 місяці (стандарт моніторингу)
- Самоконтроль глюкози вдома за схемою лікаря
- Раз на рік: огляд офтальмолога, перевірка стоп, ACR у сечі, креатинін
- Не припиняйте поточні препарати самостійно
{% else %}
- Контроль HbA1c і глюкози натще раз на рік
{% if bmi and bmi >= 25 %}- Зниження ваги на 5-7% драматично знижує ризик{% endif %}
- 30 хвилин ходьби/день мінімум
- Зменшити цукор і швидкі вуглеводи
- Висипатися — недосип погіршує толерантність до глюкози
- Контроль стресу — кортизол піднімає глюкозу
{% endif %}
