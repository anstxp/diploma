{% if had_stroke %}
У вашому профілі вказаний перенесений інсульт, і поточна модель показує **підвищений ризик повторної події** ({{ p_stroke }}%). Це означає, що **вторинна профілактика потребує перегляду** — після першого інсульту ризик другого значно вищий.

{% if on_oral_anticoagulants %}Ви приймаєте антикоагулянти — продовжуйте регулярно, **не припиняйте самостійно**.{% endif %}
{% elif has_cardiovascular_disease %}
У вашому профілі є серцево-судинне захворювання, і модель показує **підвищений ризик інсульту** ({{ p_stroke }}%, ×{{ ratio_stroke }}). ССЗ + цей профіль = підвищена увага до контролю АТ, ритму серця і ліпідів.
{% else %}
Виявлено **підвищений ризик інсульту** ({{ p_stroke }}%, ×{{ ratio_stroke }}). Більшість інсультів запобіжні через контроль артеріального тиску, мерехтіння передсердь і ліпідів.
{% endif %}

actions:
{% if had_stroke %}
- **Невролог або кардіолог — позаплановий візит**
- ЕКГ і Holter — виявити фібриляцію передсердь якщо ще не діагностовано
- Доплер сонних артерій — повторно якщо давно не робили
- Жорсткий АТ-контроль — мета <130/80
- LDL <55 мг/дл (вторинна профілактика після інсульту)
- **Продовжуйте антитромбоцитарну/антикоагулянтну терапію** регулярно
{% elif has_cardiovascular_disease %}
- **Кардіолог або невролог протягом 2-4 тижнів**
- ЕКГ і Holter — виявити фібриляцію передсердь
- Доплер сонних артерій
- Жорсткий АТ-контроль — мета <130/80
- Контроль ліпідів і глюкози
- Якщо є фібриляція передсердь — обговоріть антикоагулянти
{% else %}
- **Кардіолог або невролог протягом 2-4 тижнів**
- ЕКГ — виявити мерехтіння передсердь (фібриляцію)
- Доплер сонних артерій — атеросклероз
- Жорсткий контроль АТ — мета <130/80
- Якщо є фібриляція передсердь — антикоагулянти за призначенням
- Обмеження алкоголю, відмова від куріння
- Контроль глюкози і ліпідів
{% endif %}
