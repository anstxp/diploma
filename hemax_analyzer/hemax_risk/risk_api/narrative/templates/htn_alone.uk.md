{% if on_acei_arb or on_antihypertensives %}
Ви приймаєте антигіпертензивну терапію, поточний модельний ризик помірний ({{ p_htn }}%). **Це означає, що терапія тримає АТ під контролем** — без неї цифри були б значно вищими.

{% if sbp and sbp < 130 %}Поточний АТ {{ sbp }}/{{ dbp }} у цілі — продовжуйте поточну схему.{% elif sbp and sbp >= 130 and sbp < 140 %}АТ {{ sbp }}/{{ dbp }} близько до цілі — обговоріть з лікарем чи потрібна корекція.{% elif sbp and sbp >= 140 %}АТ {{ sbp }}/{{ dbp }} вище цілі на тлі терапії — обговоріть посилення.{% endif %}
{% elif has_hypertension %}
У вашому профілі вказана гіпертонія, але поточний модельний ризик помірний ({{ p_htn }}%). Якщо ви тримаєте АТ під контролем без ліків — це **результат вашої роботи**, продовжуйте.
{% else %}
Виявлено **помірно підвищений ризик артеріальної гіпертензії** ({{ p_htn }}%, ×{{ ratio_htn }}). Інші серцево-судинні ризики у нормі — це обнадійливо.

На цьому етапі інтервенції особливо ефективні: спосіб життя може повністю утримувати АТ у нормі **без ліків**.
{% endif %}

actions:
{% if on_acei_arb or on_antihypertensives %}

- Зменшіть сіль і алкоголь
- Контрольний візит до лікаря раз на 3-6 місяців
{% if on_acei_arb %}- Перевірте калій сироватки раз на 6-12 місяців{% endif %}
{% else %}
- АТ домашнім тонометром раз на 1-2 тижні
- Зменшіть сіль і алкоголь
- Регулярна активність 150 хв/тиждень
- Контроль ваги {% if bmi and bmi >= 25 %}— зайва вага піднімає АТ{% endif %}
- Кинути паління, якщо курите
- Контрольний візит до сімейного лікаря раз на рік
{% endif %}
