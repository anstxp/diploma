{% if has_diabetes %}

Це **не означає невдачу** — діабет це маратон, і корекція схеми лікування час від часу нормальна.

{% if takes_metformin %}Ви приймаєте метформін — обговоріть з ендокринологом чи потребує доза підвищення, чи варто додати другий препарат.{% elif on_oral_diabetics %}Ви приймаєте оральні препарати від діабету — обговоріть з ендокринологом перегляд схеми.{% elif on_insulin %}Ви на інсуліні — обговоріть з ендокринологом титрування дози.{% endif %}
{% elif hba1c and hba1c >= 6.5 %}
Ваші лабораторні показники (HbA1c {{ hba1c }}%{% if glucose %}, глюкоза {{ glucose }} mg/dL{% endif %}) **відповідають критеріям ADA для можливого діагнозу цукрового діабету 2-го типу** (HbA1c ≥6.5% або glucose ≥126 mg/dL натще). Це **не остаточний діагноз** — для підтвердження потрібен повторний аналіз і оцінка лікарем у клінічному контексті.
{% elif hba1c and hba1c >= 5.7 %}
Ваші показники (HbA1c {{ hba1c }}%) **у переддіабетичному діапазоні** (5.7-6.4%). Без втручання у 30-50% таких людей розвинеться діабет за наступні роки.
{% else %}
Ваші показники свідчать про **підвищений ризик цукрового діабету 2-го типу** ({{ p_diabetes }}%, ×{{ ratio_diabetes }} від популяційного середнього).
{% endif %}

{% if not has_diabetes %}Хороша новина: **ризик діабету 2-го типу часто можна суттєво знизити** змінами способу життя. actions:
{% if has_diabetes %}
- **Покажіть результати ендокринологу** — обговоріть посилення терапії або корекцію дози
- Самоконтроль глюкози (натще + після їжі) за схемою лікаря
- Раз на рік: огляд офтальмолога, перевірка стоп, мікроальбумін у сечі, креатинін
- Перегляньте дієту і фізичну активність — часто це впливає більше за зміну ліків
{% if bmi and bmi >= 25 %}- Зниження ваги навіть на 5% значно покращує глікемію{% endif %}
- Не припиняйте поточні препарати самостійно
{% else %}
- **Зверніться до сімейного лікаря або ендокринолога**
- Повторити HbA1c і глюкозу натще через 6-8 тижнів{% if hba1c and hba1c >= 6.5 %} для підтвердження діагнозу{% endif %}
- Розгляньте OGTT (оральний глюкозо-толерантний тест) для точної діагностики
{% if bmi and bmi >= 25 %}- Зниження ваги на 5-7% знижує ризик діабету на ~58%{% endif %}
- 150 хв помірної аеробної активності/тиждень
- Зменшити швидкі вуглеводи (білий хліб, солодкі напої, випічка)
- Збільшити клітковину (овочі, бобові, цільнозернові)
- Розгляньте сніданки з білком + жирами замість вуглеводів
{% endif %}
