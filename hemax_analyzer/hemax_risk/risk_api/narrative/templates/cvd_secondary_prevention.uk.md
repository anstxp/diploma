# Вторинна профілактика серцево-судинних подій
icon: ❤️‍🩹
severity: medium
---
У вашому профілі вже є серцево-судинне захворювання (перенесений інфаркт, ІХС{% if had_stroke %}, перенесений інсульт{% endif %}{% if has_chf %}, серцева недостатність{% endif %}). Це означає що ви знаходитесь у режимі **вторинної профілактики** — мета якої не «зменшити ризик з нуля», а **запобігти повторній події**.

**Важливо розуміти:** низькі цифри ризику у моделі ({{ p_chd | default('—') }}% для ІХС) — це **наслідок того, що терапія працює**, а не що ризик зник. У пацієнтів з відомими ССЗ повторна подія в найближчі 10 років статистично відбувається в **15-20%** випадків без терапії; ваша поточна схема знижує цю цифру у 2-3 рази.

{% if on_statins %}**Статин:** утримує атеросклеротичні бляшки стабільними. Цільові значення LDL для вашої групи **<55 мг/дл** (≤1.4 ммоль/л).{% if ldl and ldl > 55 %} Ваш поточний LDL ({{ ldl }} мг/дл) трохи вище за ціль — обговоріть з кардіологом посилення дози або додавання езетимібу.{% endif %}{% endif %}
{% if on_acei_arb %}
**ACE-інгібітор / БРА:** знижує ремоделювання міокарда після інфаркту, захищає нирки.{% endif %}
{% if on_oral_anticoagulants %}
**Антиагрегант (аспірин) або антикоагулянт:** запобігає тромбозу у вже ушкоджених судинах.{% endif %}
{% if has_diabetes %}
**Діабет + ССЗ — особливо ризикова комбінація.** Цільовий HbA1c для вас зазвичай ≤7%; LDL ≤1.4 ммоль/л (як «дуже високий ризик»).{% endif %}
---
actions:
- **Не припиняйте поточну терапію самостійно** — кожна доза має значення
- Регулярні візити до кардіолога — раз на 6 місяців або частіше за призначенням
- Контроль артеріального тиску вдома (цільове <130/80)
- Ліпідограма раз на 6-12 місяців
{% if on_statins %}- Здавайте АЛТ і креатинкіназу — для моніторингу побічних ефектів статинів{% endif %}
- Дієта середземноморського типу, обмеження солі та насичених жирів
- Аеробна активність 150 хв/тиждень (узгодити інтенсивність з кардіологом)
{% if has_diabetes %}- Раз на рік: огляд офтальмолога, перевірка стоп, мікроальбумін у сечі{% endif %}
- Не пропускайте дози — облік прийому ліків через тижневу касетницю чи додаток
---
red_flags:
- **Біль у грудях**, особливо при навантаженні або в спокої > 5 хв — терміново до лікаря або **103/112**
- Сильна задишка, нічна задишка, набряки ніг — ознаки декомпенсації
- Раптова слабкість в руці/нозі, перекос обличчя, порушення мови — **підозра на інсульт, негайно 103/112**
- Серцебиття, відчуття перебоїв, запаморочення — потребує оцінки
{% if on_oral_anticoagulants %}- Кров у сечі, чорний стілець, рясні гематоми — кровотеча на тлі антикоагулянтів{% endif %}
{% if on_statins %}- Сильні м'язові болі або потемніння сечі — можливий рабдоміоліз від статинів{% endif %}
