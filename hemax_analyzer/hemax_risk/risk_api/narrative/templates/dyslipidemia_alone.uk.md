{% if on_statins %}
Ви приймаєте статини, поточна ліпідограма у моделі видає помірний ризик ({{ p_high_chol }}%). **Це означає, що терапія працює, але цільові значення ще можна покращити** — особливо якщо у вас є інші серцево-судинні фактори ризику.

{% if has_diabetes %}У вас діабет — цільові значення LDL **жорсткіші** (зазвичай <70 мг/дл при високому ризику, <55 при дуже високому). Поточний рівень може бути недостатньо низьким.{% endif %}
{% elif has_high_chol %}
У вашому профілі вказана гіперхолестеринемія. Поточна ліпідограма стабільна (модель видає {{ p_high_chol }}%), але є простір для покращення — особливо за наявності інших факторів ризику.
{% else %}
Помірно підвищений ризик гіперхолестеринемії ({{ p_high_chol }}%). На цьому етапі **дієта і активність дають найбільший виграш**.
{% endif %}

- Повторіть ліпідограму через 3-6 місяців

- Повторіть ліпідограму натще через 3 місяці

- Замініть насичені жири на ненасичені (риба, оливкова олія, горіхи)
- 150 хв/тиждень активності
- Обмежте трансжири і фастфуд
