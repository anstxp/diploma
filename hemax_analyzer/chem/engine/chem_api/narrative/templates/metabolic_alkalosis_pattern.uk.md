# Метаболічний алкалоз
icon: ⚡
severity: low
---
Ваш бікарбонат ({{ bicarbonate }} ммоль/л) підвищений. Це ознака **метаболічного алкалозу** — олуження крові.

Часті причини:
- **Тривале блювання** — втрата соляної кислоти
- **Прийом діуретиків** (особливо петлевих, тіазидних)
- **Гіпокалемія**
- Надмірне споживання антацидів (бікарбонату)
- Гіперальдостеронізм

- Достатньо пийте воду
- Якщо є блювання — зверніться до лікаря для з'ясування причини
