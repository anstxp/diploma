# Підвищена сечова кислота
icon: 🦴
severity: low
---
Ваша сечова кислота ({{ uric_acid }} мг/дл) вище норми. Це **гіперурикемія** — фактор ризику **подагри** та сечокам'яної хвороби.

{% if on_diuretics %}
**Тіазидні діуретики (гіпотіазид, індапамід) — одна з найчастіших причин гіперурикемії** у дорослих. Тіазиди підвищують сечову кислоту приблизно у 50% пацієнтів. Це не означає що треба відміняти — у багатьох випадках перевага від лікування АТ переважує цей ризик. Але якщо вже були приступи подагри — обговоріть альтернативу з лікарем.
{% elif has_ckd %}
У вас хронічна хвороба нирок — нирки виводять сечову кислоту, тому при ХХН рівень закономірно зростає. Самостійним лікуванням це **не лікується** — потрібен контроль ХХН.
{% elif has_cardiovascular_disease %}
У вас є серцево-судинне захворювання — гіперурикемія додатково підвищує серцево-судинний ризик і часто супроводжує метаболічний синдром.
{% else %}
Не у всіх з підвищеною сечовою кислотою розвивається подагра — це залежить від генетики, способу життя, діуретиків. Але **підвищена сечова кислота сама по собі** також пов'язана з підвищеним серцево-судинним ризиком.
{% endif %}
---
actions:
{% if on_diuretics %}
- **Якщо у вас були гострі приступи подагри** — обговоріть з лікарем альтернативу (наприклад, лозартан АРБ, який знижує сечову кислоту)
- Не припиняйте діуретик самостійно
- Здайте креатинін, повторно сечову кислоту через місяць
- Раціон: обмежте червоне м'ясо, морепродукти, пиво
- Достатньо пийте воду (2-3 л/день)
{% elif has_ckd %}
- Покажіть результати нефрологу
- Якщо вже були **напади подагри**, є тофуси, частий уролітіаз або сечова кислота >9-10 мг/дл при ХХН ≥3 стадії — може потребуватися алопуринол
- **Без симптомів подагри** ULT (urate-lowering therapy) зазвичай не починають — за рекомендаціями ACR
- Контроль ХХН як основа лікування
{% else %}
- Зверніться до сімейного лікаря для оцінки контексту
- Здайте: креатинін (нирки виводять сечову кислоту), HbA1c, ліпідний профіль
- Раціон: обмежте червоне м'ясо, морепродукти, субпродукти, пиво, солодкі напої з фруктозою
- Достатньо пийте воду (2-3 л/день)
- Якщо є зайва вага — її зниження допомагає
- Якщо приймаєте діуретики — обговоріть з лікарем
{% endif %}
---
red_flags:
- **Гострий біль і набряк суглоба** (особливо великого пальця ноги) — можливий гострий приступ подагри
- Біль у попереку, кров у сечі — можливі камені нирок
{% if has_cardiovascular_disease %}
- Гострий біль у грудях — гіперурикемія посилює серцево-судинний ризик
{% endif %}
