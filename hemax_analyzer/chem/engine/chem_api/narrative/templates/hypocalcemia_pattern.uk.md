# Знижений кальцій
icon: 🦴
severity: medium
---
Ваш кальцій ({{ calcium }} мг/дл) нижче норми. Гіпокальцемія може спричиняти м'язові судоми, поколювання, у тяжких випадках — судоми загального характеру.

Можливі причини:
- **Дефіцит вітаміну D** — найчастіша причина
- **Гіпопаратиреоз** — недостатність паратгормону
- **Хронічна хвороба нирок**
- **Низький альбумін** — артефакт (загальний кальцій занижений, а іонізований нормальний)
- **Прийом певних ліків** (бісфосфонати, протисудомні)
---
actions:
- Зверніться до лікаря для з'ясування причини
- Здайте: альбумін, вітамін D (25-OH), фосфат, магній, ПТГ
- Якщо альбумін низький — кальцій може бути "штучно" заниженим, тоді важливий іонізований кальцій
- Не приймайте кальцій у великих дозах без призначення
- Достатньо вживайте молочних продуктів, темно-зеленої зелені
---
red_flags:
- Поколювання губ, пальців
- М'язові судоми (особливо в литках, кистях)
- Серцебиття
- Судоми загального характеру — **негайно** в приймальне
