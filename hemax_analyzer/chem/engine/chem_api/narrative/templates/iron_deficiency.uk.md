# Дефіцит заліза
icon: 🩸
severity: medium
---
{% if ferritin %}Ваш феритин ({{ ferritin }} нг/мл){% else %}Ваші показники заліза{% endif %} вказують на **дефіцит заліза**. Феритин — найкращий маркер запасів заліза в організмі.{% if tsat %} Низький TSAT ({{ tsat }}%) це підтверджує.{% endif %}

Дефіцит заліза розвивається коли витрати перевищують надходження:
{% if is_female and age and age < 50 %}
- У жінок репродуктивного віку — найчастіше через **рясні або тривалі місячні**
- Інша поширена причина — недостатнє надходження з їжею (особливо при вегетаріанських дієтах)
{% elif is_male or (age and age >= 50) %}
- У чоловіків і жінок старше 50 — **завжди потребує з'ясування причини**
- Найчастіше — приховані крововтрати з ШКТ (виразки, поліпи, рідше пухлини)
- Друга причина — недостатнє засвоєння заліза з їжі
{% endif %}

Симптоми: втома, задишка при навантаженнях, блідість, ламкі нігті, випадіння волосся, відчуття холоду в кінцівках.
{% if on_oral_anticoagulants %}

**Особливо важливо для вас:** ви приймаєте антикоагулянти. Поєднання залізодефіциту з антикоагулянтами часто вказує на **приховану крововтрату з ШКТ** — це потребує термінового обстеження.
{% endif %}
---
actions:
- Зверніться до сімейного лікаря з цими результатами
- Здайте загальний аналіз крові — для оцінки розвитку анемії
- {% if is_male or (age and age >= 50) %}**Обов'язково обговоріть гастроскопію/колоноскопію** для виключення джерела крововтрати з ШКТ{% else %}Якщо місячні рясні — обговоріть це з гінекологом{% endif %}
- Не приймайте залізо самостійно — лікар призначить дозу і форму
- Раціон: червоне м'ясо, печінка, бобові, листова зелень з продуктами багатими на вітамін С
- Уникайте чаю і кави з їжею (вони знижують засвоєння)
{% if on_oral_anticoagulants %}- Не змінюйте дозу антикоагулянтів самостійно — навіть тимчасово{% endif %}
---
red_flags:
- Сильна задишка, серцебиття, запаморочення
- Чорний дьогтеподібний кал або кров у стільці — **негайно** до лікаря
- Кров у блювотинні
- Сильна втома, що погіршується
{% if on_oral_anticoagulants %}- Будь-яка нова кровотеча (носова, ясна, синці) на тлі прийому антикоагулянтів — до лікаря{% endif %}
