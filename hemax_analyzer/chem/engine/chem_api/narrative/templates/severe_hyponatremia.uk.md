# Критично низький натрій
icon: 🔴
severity: high
---
Ваш натрій ({{ sodium }} ммоль/л) **критично низький**. При значеннях ≤125 ммоль/л можливі судоми, набряк мозку, кома. Це **невідкладний стан**.

{% if on_diuretics %}
**Найімовірніша причина — діуретики**, особливо тіазидні (гіпотіазид, індапамід). Тіазиди — **#1 причина медикаментозної гіпонатріємії**, особливо у літніх жінок. Препарат не «поганий» — просто конкретно вам ця доза/тип не підходить.
{% elif has_cardiovascular_disease %}
У вас є серцево-судинна хвороба — серцева недостатність часто супроводжується гіпонатріємією через активацію антидіуретичної системи. Декомпенсація СН — часта причина.
{% elif has_liver_disease %}
У вас є хвороба печінки — цироз з асцитом часто супроводжується гіпонатріємією через перерозподіл рідини і затримку води. Це маркер декомпенсації.
{% else %}
Причини: SIADH (синдром неадекватної секреції антидіуретичного гормону), важка серцева чи печінкова недостатність, надмірне вживання води, прийом діуретиків, гіпотиреоз, наднирникова недостатність.
{% endif %}
---
actions:
- **Зверніться у приймальне відділення сьогодні**
- Не пийте велику кількість води до огляду
- Принесіть список ліків, особливо діуретиків
- Можлива потреба у госпіталізації для контрольованого підвищення натрію
- **Швидка корекція натрію може бути небезпечною** — лікування потребує ретельного моніторингу
{% if on_diuretics %}
- **Не припиняйте діуретик самостійно** — у приймальному вирішать чи відмінити, замінити, чи зменшити дозу
- Скажіть лікарю точну назву препарату і дозу — це критично для тактики
{% endif %}

red_flags:
- Сильний головний біль, нудота, блювання
- Сплутаність свідомості, дезорієнтація
- М'язові судоми або посмикування
- Сонливість, зниження реакції
- Втрата свідомості, судоми — **виклик швидкої**
