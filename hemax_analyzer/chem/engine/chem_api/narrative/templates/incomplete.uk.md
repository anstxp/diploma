# Неповний аналіз
icon: ⚠️
severity: low
---
У вашому аналізі не вистачає ключових показників для повної інтерпретації. Можливо, ви ввели не всі значення з бланку, або це частковий аналіз.

Для повноцінної інтерпретації біохімії потрібні: глюкоза або HbA1c, креатинін, АЛТ або АСТ, ліпідна панель (загальний холестерин, HDL, тригліцериди).
---
actions:
- Перевірте, чи всі значення введені коректно
- Якщо це скан лабораторного результату — переконайтесь, що документ повний
- Здайте відсутні показники для повної картини
