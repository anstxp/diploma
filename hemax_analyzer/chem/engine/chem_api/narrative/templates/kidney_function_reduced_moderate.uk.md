# Знижена функція нирок
icon: 🫘
severity: medium
---
Ваш eGFR ({{ egfr }} мл/хв/1.73м²) знижений, креатинін {{ creatinine }} мг/дл підвищений. Це **знижена функція нирок** на момент аналізу. Сам по собі один аналіз ще не означає хронічну хворобу нирок (ХХН) — за класифікацією KDIGO, ХХН встановлюється коли порушення функції нирок (ШКФ <60) **зберігається понад 3 місяці**. Тому важливо також виключити **гостре** погіршення (зневоднення, недавні ліки, інфекція) і повторити аналіз через 4-12 тижнів.

{% if has_ckd %}
У вашому профілі вже вказана ХХН — поточні показники дозволяють відстежити динаміку. Якщо це **погіршення відносно ваших звичних значень** — потрібен позаплановий візит до нефролога. Якщо в межах вашого стабільного діапазону — це частина моніторингу.
{% elif has_diabetes %}
У вас цукровий діабет — це **одна з найчастіших причин** хронічного ураження нирок (діабетична хвороба нирок). Але для встановлення цього діагнозу потрібні додаткові дані: **співвідношення альбумін/креатинін у сечі (ACR)**, повторний креатинін у динаміці, оцінка тривалості діабету, виключення інших причин (гіпертонія, ліки, обструкція). На цій стадії, якщо це справді діабетична нефропатія, її можна стабілізувати — головне інструменти: жорсткий контроль глюкози (HbA1c часто <7%), АТ <130/80, при підтвердженні альбумінурії — АСЕ-інгібітор/АРБ і SGLT2-інгібітори.
{% elif has_hypertension %}
У вас гіпертонія, яка є другою за поширеністю причиною ХХН (після діабету). Контроль АТ <130/80 — критично важливий щоб запобігти прогресуванню.
{% elif on_acei_arb %}
Ви приймаєте АСЕ-інгібітор/АРБ — ці препарати **захищають нирки** при ХХН, але можуть спричинити невелике зростання креатиніну (до 30%) на старті терапії. Це **очікувано і не привід припиняти**. Але потрібно переконатися що це не гостре погіршення.
{% else %}
ХХН частіше за все спричинена діабетом, гіпертонією, тривалим прийомом певних ліків (наприклад, НПЗП), віковими змінами. На цій стадії повне відновлення можливе при усуненні причини, або принаймні припинення прогресування.
{% endif %}
---
actions:
{% if has_ckd %}
- **Покажіть результати нефрологу** — порівняйте з вашою звичайною динамікою
- Перевірте чи приймаєте всі призначені препарати (АСЕ-інг/АРБ, статини)
- Уникайте НПЗП (ібупрофен, диклофенак) — вони шкодять навіть здоровим ниркам
- Контроль АТ вдома — мета <130/80 мм рт.ст.
- Перевірте калій сироватки якщо приймаєте АСЕ-інг/АРБ або калій-зберігаючі діуретики
{% elif has_diabetes %}
- **Зверніться до ендокринолога і нефролога** — для оцінки потенційної діабетичної хвороби нирок
- Здайте: HbA1c, **співвідношення альбумін/креатинін у сечі (ACR)**, повторно креатинін через 4-12 тижнів — для підтвердження стійкості і виключення гострого погіршення
- Аналіз сечі (загальний) — щоб виключити інші причини
- Якщо ACR підтверджена — обговоріть АСЕ-інгібітор/АРБ і SGLT2 (наприклад, емпагліфлозин/дапагліфлозин — вони захищають нирки у діабетиків)
- Жорсткіші цілі при підтвердженні ХХН: HbA1c <7%, АТ <130/80
- Перегляньте дози метформіну з лікарем (при низькому ШКФ потрібна корекція)
{% elif has_hypertension %}
- **Контроль АТ — головний пріоритет.** Цільовий АТ <130/80 мм рт.ст.
- Здайте: мікроальбумін у сечі, повторно креатинін через місяць
- Обговоріть з лікарем АСЕ-інгібітор або АРБ — вони і АТ знижують, і нирки захищають
- Уникайте НПЗП (ібупрофен/диклофенак) — погіршують і АТ, і нирки
{% elif on_acei_arb %}
- Покажіть результати лікарю який призначив АСЕ-інг/АРБ
- Здайте калій сироватки — АСЕ-інг можуть його підвищувати при ХХН
- **Не припиняйте препарат самостійно** — це може зашкодити більше ніж невелике зростання креатиніну
- Повторіть креатинін через 2-4 тижні щоб побачити динаміку
{% else %}
- Зверніться до сімейного лікаря або терапевта для встановлення причини
- **Повторіть креатинін через 4-12 тижнів** — за KDIGO для діагностики ХХН потрібна стійкість понад 3 місяці
- Здайте: повторно креатинін, аналіз сечі, **співвідношення альбумін/креатинін у сечі (ACR)**
- Перевірте артеріальний тиск і глюкозу — найчастіші причини ХХН
- Уникайте регулярного прийому **ібупрофену, диклофенаку** та інших НПЗП
- Перевірте чи нічого не змінилося нещодавно: нові ліки, рентген з контрастом, дегідратація — вони можуть давати **гостре** зниження
- Достатньо пийте воду, але без надлишку
{% endif %}

red_flags:
- Поява набряків
- Зменшення кількості сечі
- Кров у сечі (рожевий або коричневий колір)
- Стійка втома, слабкість
{% if on_acei_arb %}
- Сухий кашель — частий побічний АСЕ-інгібіторів, обговоріть заміну на АРБ
- М'язова слабкість, неритмічне серцебиття — можлива гіперкаліємія від АСЕ-інг
{% endif %}
