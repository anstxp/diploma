# Підвищені тригліцериди
icon: 💗
severity: low
---
Ваші тригліцериди ({{ trigly }} мг/дл) вище норми, але не критично високі. Часті причини: надмірне споживання простих вуглеводів і алкоголю, ожиріння, малорухливий спосіб життя, неконтрольований діабет.

При помірному підвищенні зміни способу життя зазвичай дають значне покращення.
---
actions:
- Повторіть аналіз натще (мінімум 12 годин голоду) — їжа значно підвищує тригліцериди
- Зменште прості вуглеводи: цукор, солодке, газовані напої, фруктові соки, біла мука
- Обмежте алкоголь (особливо пиво)
- Регулярна аеробна активність (30+ хв на день)
- Здайте додатково: HbA1c, ТТГ
- Повторіть ліпіди через 3 місяці
