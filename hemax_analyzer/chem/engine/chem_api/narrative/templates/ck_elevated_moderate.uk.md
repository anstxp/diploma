# Помірно підвищений КФК
icon: 💪
severity: low
---
Ваш КФК (CK) ({{ ck }} U/L) помірно підвищений. КФК — фермент м'язів, його рівень росте після:

- Інтенсивних тренувань, особливо силових (1-2 дні після)
- Тривалих фізичних навантажень
- Внутрішньом'язових ін'єкцій
- Прийому статинів (іноді)
- Гіпотиреозу (рідше)

При помірному підвищенні без симптомів зазвичай неcritique.
---
actions:
- Якщо нещодавно тренувалися — повторіть аналіз через 5-7 днів спокою
- Здайте: ТТГ — для виключення гіпотиреозу
- Якщо приймаєте статини і м'язові болі — зверніться до лікаря
- Достатньо пийте воду
---
red_flags:
- Сильні м'язові болі
- Темна сеча
- Слабкість
- При значеннях >1000 — повторіть найближчим часом
