# Картина гепатоцелюлярного ураження печінки
icon: 🟠
severity: medium
---
Ваші печінкові ферменти {% if alt %}АЛТ ({{ alt }} U/L){% endif %}{% if alt and ast %} та {% endif %}{% if ast %}АСТ ({{ ast }} U/L){% endif %} підвищені. Ці ферменти знаходяться всередині клітин печінки і потрапляють у кров при подразненні або ушкодженні цих клітин. Така картина називається **гепатоцелюлярним патерном** — на відміну від холестатичного, де переважає підвищення ЛФ і ГГТ.

{% if has_liver_disease %}
У вас вже відома хвороба печінки — поточні значення треба порівняти з вашими звичайними. Якщо це **значне погіршення відносно звичних значень** — терміново до гепатолога. Якщо в межах вашого діапазону — це частина моніторингу.
{% elif on_statins %}
Ви приймаєте статини — вони у ~2% пацієнтів викликають **легке-помірне підвищення АЛТ/АСТ**. Зазвичай це безпечно, не вимагає відміни і часто проходить самостійно. **Не припиняйте статини самостійно** — це підвищує ризик інфаркту.
{% elif has_diabetes %}
У діабетиків часто розвивається **неалкогольна жирова хвороба печінки (NAFLD)** — приблизно у 70% пацієнтів з діабетом 2 типу. Це найімовірніша причина у вашому випадку, особливо якщо є зайва вага.
{% else %}
Найчастіші причини:
- **Жирова хвороба печінки** (NAFLD) — особливо при надмірній вазі, цукровому діабеті
- **Вірусні гепатити** (B, C) — потребують виключення
- **Токсичне ураження** — алкоголь, певні ліки (парацетамол у великих дозах, статини, амоксицилін-клавуланат)
- Рідше — аутоімунні гепатити
{% endif %}
---
actions:
{% if has_liver_disease %}
- **Покажіть результати гепатологу** — порівняйте з вашою звичайною динамікою
- Перевірте чи приймаєте всі призначені препарати регулярно
- Відмова від алкоголю — критично важлива
- УЗД печінки якщо давно не робили
{% elif on_statins %}
- **Не припиняйте статини самостійно** — обговоріть з лікарем
- Якщо АЛТ <3× верхньої межі норми — статини зазвичай безпечні
- Якщо АЛТ ≥3× верхньої межі — лікар може тимчасово зменшити дозу або замінити статин
- Здайте через 4 тижні: АЛТ, АСТ, ГГТ, КК (креатинкіназа) — щоб виключити рабдоміоліз
{% elif has_diabetes %}
- Зверніться до ендокринолога або гастроентеролога
- Здайте: HBsAg, anti-HCV (виключити вірусні), повторно АЛТ/АСТ
- УЗД печінки — оцінити жировий гепатоз
- Контроль HbA1c і ваги — основа лікування NAFLD
- Уникайте алкоголю
{% else %}
- Зверніться до сімейного лікаря або гастроентеролога
- Здайте: HBsAg, anti-HCV (вірусні гепатити), повторно АЛТ/АСТ через 4 тижні
- Якщо AST/ALT > 2 — виключіть зловживання алкоголем
- Обмежте алкоголь до **повної відмови** до наступних аналізів
- Перегляньте список ліків з лікарем
- УЗД печінки — для оцінки структури
{% endif %}

red_flags:
- Жовтушність шкіри або очей (іктер)
- Потемніння сечі або знебарвлення калу
- Сильний свербіж шкіри
- Біль у правому підребер'ї
- Сплутаність свідомості — **негайно** в приймальне
{% if on_statins %}
- Сильні м'язові болі або потемніння сечі — **можливий рабдоміоліз від статинів** — терміново до лікаря
{% endif %}
