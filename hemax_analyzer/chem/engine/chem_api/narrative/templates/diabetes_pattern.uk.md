Це **не означає невдачу** — діабет це маратон, і коригувати схему лікування час від часу нормально.{% endif %}{% if a1c_at_target %}HbA1c у цільовому діапазоні — глікемія під контролем. Стандартна ціль для більшості діабетиків: HbA1c <7%, глюкоза натще 4.4–7.2 ммоль/л (80–130 мг/дл). Точна ціль залежить від віку, тривалості діабету та супутніх станів — обговоріть з ендокринологом.{% endif %}{% if not_known_diabetic %}{% if a1c %}Ваш HbA1c — {{ a1c }}% — у діабетичному діапазоні (≥6.5%).{% endif %}{% if glucose %} Глюкоза натще {{ glucose }} мг/дл також підвищена.{% endif %} Це **дуже ймовірний цукровий діабет**, але офіційний діагноз потребує підтвердження повторним аналізом або іншим тестом (OGTT).

HbA1c відображає **середній рівень цукру за 2-3 місяці**, тому він стабільніший за окремий вимір глюкози. Сучасний діабет — це керована хвороба: при правильному лікуванні люди живуть повноцінне життя, без ускладнень.{% endif %}

actions:
{% if has_diabetes %}
- **Покажіть результати ендокринологу** на найближчому плановому прийомі (раніше — якщо HbA1c >8%)
- Самоконтроль глюкози (натще + після їжі) за схемою лікаря
- Перегляньте дієту і фізичну активність — часто допомагає більше ніж зміна ліків
- Раз на рік: огляд офтальмолога, перевірка стоп, мікроальбумін у сечі, креатинін
- Якщо є зайва вага — навіть зниження на 5% значно покращує глікемію
{% endif %}
{% if not_known_diabetic %}
- **Зверніться до ендокринолога або сімейного лікаря у найближчі 1-2 тижні**
- Повторіть HbA1c, глюкозу натще або OGTT — за рекомендацією лікаря (ADA приймає кілька варіантів підтвердження, точний вибір залежить від клінічної ситуації)
- Перевірте функцію нирок (креатинін, мікроальбумін у сечі), очі (огляд офтальмолога)
- Перегляньте раціон: зменшіть швидкі вуглеводи, цукор, газовані напої
- Почніть регулярні помірні фізичні навантаження (30 хв/день)
- Якщо є зайва вага — обговоріть з лікарем план зниження
{% endif %}

- Запах ацетону з рота, нудота, біль у животі — можливий кетоацидоз — **негайно** в приймальне
- Туман або погіршення зору
- Незагойні рани на ногах
{% if has_diabetes %}
- Часта гіпоглікемія (тремтіння, холодний піт, голод) — потребує корекції доз
{% endif %}
