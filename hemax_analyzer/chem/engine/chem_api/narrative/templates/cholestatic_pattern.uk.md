# Картина холестазу — застою жовчі
icon: 🟠
severity: medium
---
Ваші ферменти **{% if alp %}ЛФ ({{ alp }} U/L){% endif %}{% if alp and ggt %} та {% endif %}{% if ggt %}ГГТ ({{ ggt }} U/L){% endif %}** підвищені, а АЛТ/АСТ нормальні або трохи підвищені. Це **холестатична картина** — порушення відтоку жовчі.

{% if pregnant %}
**Під час вагітності легке-помірне підвищення ЛФ — фізіологічне** (плацента виробляє свій ізоензим ЛФ). Тому сам по собі цей результат у вагітних часто не означає патології. Але якщо ЛФ зросла різко або підвищилась з ГГТ і свербежем — це може бути **холестаз вагітних**, який потребує спостереження.
{% elif on_oral_anticoagulants %}
Ви приймаєте антикоагулянти — деякі з них (особливо варфарин, рідше DOACи) можуть спричиняти **лікарський холестаз**. Це треба з'ясувати — препарат може потребувати заміни.
{% elif has_liver_disease %}
У вас вже є відома хвороба печінки — поточна картина може означати прогресування або нову проблему (наприклад, біліарну обструкцію). Терміново до гастроентеролога/гепатолога.
{% else %}
Можливі причини:
- **Жовчні камені** — закупорка протоків
- **Лікарська холестатична жовтяниця** — деякі ліки
- **Первинний біліарний холангіт** (PBC) — частіше у жінок
- При ізольованому ГГТ — часто алкоголь
- Рідше — пухлини
{% endif %}
---
actions:
{% if pregnant %}
- **Обговоріть з акушером-гінекологом** — у вагітних потрібен ретельніший підхід
- Якщо є свербіж (особливо долонь і стоп) — треба здати **жовчні кислоти сироватки** для діагностики холестазу вагітних
- УЗД печінки — безпечне у вагітних, варто зробити
- Не приймайте жодних препаратів без консультації з лікарем
{% elif on_oral_anticoagulants %}
- **Зверніться до лікаря який призначив антикоагулянт** — обговоріть можливу заміну
- Не припиняйте препарат самостійно — це підвищує ризик тромбозу
- Здайте: повторно ЛФ/ГГТ через 2-4 тижні після можливої зміни препарату, УЗД печінки
- Перевірте інші ваші препарати — можливо є ще винуватці
{% else %}
- Зверніться до гастроентеролога
- Здайте: повторно ЛФ/ГГТ, **УЗД печінки і жовчних шляхів**, білірубін
- Якщо ізольовано ГГТ підвищений — перегляньте споживання алкоголю
- Перегляньте з лікарем список ліків
- Наступні тести (за призначенням): MRCP або CT якщо є підозра на блок жовчних шляхів
{% endif %}

- Свербіж по всьому тілу
- Жовтушність шкіри/очей
- Біль у правому підребер'ї, особливо після жирної їжі
- Знебарвлений кал, темна сеча
- Гарячка з ознобом
{% if pregnant %}
- Сильний свербіж долонь/стоп вночі — підозра на холестаз вагітних, **до акушера ASAP**
- Зменшення рухів плода — терміново до акушера
{% endif %}
