# Знижений магній
icon: 🦴
severity: low
---
Ваш магній ({{ magnesium }} мг/дл) нижче норми. Дефіцит магнію часто супроводжує гіпокалемію і гіпокальцемію (важко скоригувати останні без корекції магнію).

{% if on_diuretics %}
**Найімовірніша причина — діуретики**, особливо петлеві (фуросемід, торасемід) і тіазидні. Вони активно виводять магній з сечею. У більшості пацієнтів на діуретиках магній періодично потребує добавки.
{% elif on_oral_diabetics %}
У пацієнтів на оральних препаратах від діабету (особливо метформіні) магній часто знижується через знижене всмоктування у кишечнику і підвищене виведення з сечею при глюкозурії.
{% elif has_liver_disease %}
У хворих на печінкові захворювання (особливо при хронічному алкоголізмі) дефіцит магнію — поширене явище і може погіршувати загальний стан.
{% else %}
Причини: дефіцит у раціоні, мальабсорбція, тривала діарея, прийом діуретиків чи інгібіторів протонної помпи (ІПП), хронічний алкоголізм.
{% endif %}
---
actions:
{% if on_diuretics %}
- Обговоріть з лікарем — можливо потрібна добавка магнію орально (наприклад, магнію цитрат) або корекція діуретика
- Здайте: калій, кальцій (часто всі троє знижені разом)
- Не припиняйте діуретик самостійно
- Більше горіхів, насіння, темно-зеленої зелені, шоколаду в раціоні
{% elif has_liver_disease %}
- Зверніться до гастроентеролога — часто потрібна добавка магнію плюс цинк
- Якщо вживаєте алкоголь — повна відмова покращить багато показників
- Здайте: калій, кальцій, фосфор
{% else %}
- Здайте: калій, кальцій, креатинін
- Перевірте з лікарем список ліків (особливо ІПП — омепразол, пантопразол)
- Більше горіхів, насіння, темно-зеленої зелені, шоколаду в раціоні
- Якщо є виражений дефіцит — лікар призначить препарати магнію
- Не приймайте великі дози магнію самостійно при ХХН
{% endif %}

red_flags:

- Слабкість
- Тремор

{% endif %}
