# Phosphate disturbance
icon: 🦴
severity: low
---
{% if phosphate %}Your phosphate ({{ phosphate }} mg/dL) is outside the normal range.{% endif %} Rarely critical alone, but context matters — phosphorus often shifts together with calcium and PTH.

**High phosphate** common causes: CKD, hypoparathyroidism, excessive phosphate intake (phosphoric acid sodas).

**Low phosphate** — undernutrition (refeeding syndrome), chronic alcoholism, hyperparathyroidism, antacid use.
