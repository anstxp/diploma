# Порушення рівня фосфату
icon: 🦴
severity: low
---
{% if phosphate %}Ваш фосфат ({{ phosphate }} мг/дл) поза межами норми.{% endif %} Самостійно це рідко критично, але контекст важливий — фосфор часто змінюється разом з кальцієм та паратиреоїдним гормоном.

При **підвищеному фосфаті** часті причини: хронічна хвороба нирок, гіпопаратиреоз, надмірне споживання фосфатних добавок (газовані напої з фосфорною кислотою).

При **зниженому фосфаті** — недоїдання (refeeding syndrome), хронічний алкоголізм, гіперпаратиреоз, прийом антацидів.

- При зниженому фосфаті — переглянути харчування
- При підвищеному + ХХН — потрібен дієтолог
