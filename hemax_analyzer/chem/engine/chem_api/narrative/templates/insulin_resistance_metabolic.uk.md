{% if has_diabetes %}
Поєднання змін у вашому аналізі — підвищені тригліцериди, знижений HDL{% if alt %}, помірно підвищений АЛТ ({{ alt }} U/L){% endif %} — типове для **діабетичного метаболічного синдрому**. Це не «передує діабету» (у вас вже діагноз), але вказує що **метаболічний компонент потребує уваги** на тлі основної терапії.

{% if on_statins %}Ви приймаєте статини — це знижує ризик серцево-судинних подій, але мало впливає на тригліцериди та HDL. Якщо TG залишаються високими, обговоріть з кардіологом омега-3 у високих дозах або фібрати.{% endif %}

У діабетиків характерне підвищення тригліцеридів і зниження HDL — це **частина клінічної картини**, але вимагає окремої уваги для зниження серцево-судинного ризику.
{% else %}
Поєднання змін у вашому аналізі — підвищені тригліцериди, знижений HDL{% if alt %}, помірно підвищений АЛТ ({{ alt }} U/L){% endif %} — типове для **інсулінорезистентності та неалкогольної жирової хвороби печінки (NAFLD)**. Часто це передує діабету і метаболічному синдрому.

Це не "хвороба з симптомами", а **обмінне порушення**, яке поступово розвивається при надлишковій масі, малорухливому способі життя та надмірному споживанні цукру/рафінованих вуглеводів.
{% endif %}

actions:
{% if has_diabetes %}
- Обговоріть результати з ендокринологом чи кардіологом — можливо, потребує **посилення ліпідної терапії**
- Цільові значення для діабетика: **LDL <1.8 ммоль/л** (високий ризик) або **<1.4 ммоль/л** (дуже високий ризик при наявних ускладненнях)
- Регулярна активність (150 хв/тиж аеробної + 2 силові) — впливає на TG/HDL сильніше за дієту
- Якщо є зайва вага — навіть 5-7% зниження покращує всі метаболічні маркери
- Розгляньте УЗД печінки — у діабетиків NAFLD є частим супутнім станом
- Повторіть ліпідограму та печінкові ферменти через 3-6 місяців
{% if not on_statins %}- Обговоріть з лікарем призначення статинів — у діабетиків це майже завжди показано{% endif %}
{% else %}
- Зверніться до сімейного лікаря або ендокринолога для оцінки повного ліпідного й метаболічного профілю
- Додатково: HbA1c, ТТГ, феритин, УЗД печінки
- Перегляньте раціон: середземноморський тип харчування дає найкращі результати
- Регулярна активність (150 хв/тиж аеробної + 2 силові)
- Якщо є зайва вага — мета: 5-10% зниження за 6 місяців
- Обмежте алкоголь
- Повторіть аналізи через 6 місяців
{% endif %}
