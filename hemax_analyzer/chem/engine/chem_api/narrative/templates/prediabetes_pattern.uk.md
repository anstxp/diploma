{% if has_diabetes %}
У вашому профілі вже вказаний цукровий діабет. Поточні показники — {% if a1c %}HbA1c {{ a1c }}%{% endif %}{% if a1c and glucose %} і {% endif %}{% if glucose %}глюкоза натще {{ glucose }} мг/дл{% endif %} — **не вказують на нову проблему** і не означають «передіабет» (термін, що застосовується лише до людей без встановленого діагнозу).

Для діабетиків ADA рекомендує цільове **HbA1c < 7%** для більшості дорослих{% if a1c and a1c < 7.0 %} — ваше значення {{ a1c }}% у цьому діапазоні, **контроль адекватний**.{% elif a1c and a1c < 7.5 %} — ваше значення {{ a1c }}% **наближається до верхньої межі**, але ще не виходить за неї.{% else %} — ваше значення вище за ціль і потребує посилення терапії.{% endif %} У окремих випадках (молоді, без ускладнень) цільове значення може бути <6.5%, але це обговорюється індивідуально з ендокринологом.

{% if takes_metformin or on_oral_diabetics %}**Не припиняйте поточну терапію** — поточні цифри свідчать що вона працює. Якщо рівні починають дрейфувати вгору, обговоріть з ендокринологом корекцію дози.{% endif %}
{% else %}
{% if a1c %}Ваш HbA1c — {{ a1c }}% — у предіабетичному діапазоні (5.7-6.4%).{% endif %}{% if glucose %} Глюкоза натще {{ glucose }} мг/дл також на верхній межі.{% endif %}

**Це не діабет**, але це сигнал що ваш організм починає погано переробляти цукор. Без змін у способі життя у близько 30-50% людей з предіабетом протягом 5 років розвинеться повноцінний діабет. Хороші новини: це **зворотний стан** при правильних діях.
{% endif %}

actions:
{% if has_diabetes %}
- **Регулярний моніторинг HbA1c** — стандарт раз на 3 місяці
- Самоконтроль глюкози вдома за схемою лікаря
- Продовжуйте регулярні візити до ендокринолога (раз на 3-6 місяців)
- Не припиняйте поточні препарати самостійно
- Контроль інших факторів ризику: АТ, ліпіди (особливо LDL — цільові значення для діабетика жорсткіші)
- Здоровий спосіб життя — рух, дієта, нормалізація ваги — критично важливі **навіть на тлі терапії**
{% else %}
- Зменште споживання швидких вуглеводів (солодке, біла мука, газовані напої, фруктові соки)
- Збільште активність до 150 хв/тиждень помірних навантажень (швидка ходьба підходить)
- Якщо є зайва вага — навіть **5-7% зниження маси** часто повертає метаболізм у норму
- Повторіть HbA1c через 6 місяців
- Зверніться до сімейного лікаря для оцінки інших факторів ризику (АТ, ліпіди)
{% endif %}

{% else %}

{% endif %}
