# Картина гострого панкреатиту
icon: 🔴
severity: high
---
Ваша {% if lipase %}ліпаза ({{ lipase }} U/L){% else %}амілаза ({{ amylase }} U/L){% endif %} **значно підвищена**. Такі значення характерні для **гострого панкреатиту** — гострого запалення підшлункової залози.

Гострий панкреатит — невідкладний стан, який потребує негайної медичної допомоги. Найчастіші причини: жовчні камені, надмірне вживання алкоголю, високі тригліцериди.
---
actions:
- **Негайно зверніться у приймальне відділення**, навіть якщо біль слабкий

- Принесіть список ліків
- Можлива потреба в УЗД, КТ живота, госпіталізації
- Якщо є супутні високі тригліцериди — це може бути причиною
