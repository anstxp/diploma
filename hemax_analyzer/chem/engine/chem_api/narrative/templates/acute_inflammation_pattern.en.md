# Acute inflammation pattern
icon: 🦠
severity: medium
---
Your CRP ({{ crp }} mg/L) is **markedly elevated**. Values above 30 mg/L often indicate **acute inflammation** — most commonly bacterial infection, but also trauma, surgery, or severe autoimmune flare.
---
actions:
- If symptoms (fever, pain, malaise), **see a doctor today**
- If no obvious symptoms, still seek evaluation
- Get: CBC with differential, procalcitonin (for bacterial infection)
- If CRP > 100, this is a serious finding — don't delay
- Don't start antibiotics without prescription
---
red_flags:
- High fever (>38.5°C / 101.3°F) that doesn't break
- Severe headache + stiff neck
- Severe abdominal or chest pain
- Confusion
- Progressive worsening
