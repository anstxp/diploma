# Критично високий калій — небезпека для серця
icon: 🔴
severity: high
---
Ваш калій ({{ potassium }} ммоль/л) **критично високий**. При значеннях ≥6.0 ммоль/л є реальний **ризик небезпечних аритмій серця**, аж до зупинки серця. Це **невідкладний стан**.

{% if on_acei_arb and has_ckd %}
**Дуже ймовірно, причина у поєднанні**: ви приймаєте АСЕ-інгібітор/АРБ при хронічній хворобі нирок. Ці препарати підвищують калій, а нирки не можуть його ефективно вивести. **Не чекайте — у приймальне!**
{% elif on_acei_arb %}
Ви приймаєте АСЕ-інгібітор/АРБ — ці препарати **часто спричиняють гіперкаліємію**, особливо у поєднанні з іншими факторами. Це обов'язково треба врахувати у приймальному.
{% elif has_ckd %}
У вас хронічна хвороба нирок — нирки не можуть ефективно вивести надлишок калію, тому будь-який тригер (харчування, ліки) призводить до критичного підвищення.
{% elif on_diuretics %}
Ви приймаєте діуретики — якщо це калій-зберігаючі (спіронолактон, еплеренон), вони можуть бути причиною. Треба з'ясувати у приймальному.
{% else %}
Найчастіші причини: тяжка хронічна хвороба нирок, прийом певних ліків (іАПФ, БРА, спіронолактон, НПЗП), масивне руйнування клітин (травма, опіки), діабетичний кетоацидоз.
{% endif %}
---
actions:
- **Негайно зверніться у приймальне відділення** — не чекайте до завтра
- Не приймайте: добавки калію, замінники солі з калієм, банани/абрикоси/картоплю до огляду лікарем
- Принесіть з собою список усіх ліків
- Можлива потреба у ЕКГ та екстрених заходах для зниження калію
{% if on_acei_arb %}
- **Не припиняйте АСЕ-інгібітор/АРБ самостійно** — у приймальному вирішать чи відмінити, тимчасово зменшити, чи замінити
{% endif %}
{% if on_diuretics %}
- Скажіть лікарю які саме діуретики приймаєте — це критично для тактики
{% endif %}
---
red_flags:
- Серцебиття, "перебої" в роботі серця
- М'язова слабкість, особливо в ногах
- Поколювання чи оніміння
- Відчуття нерегулярного пульсу
- Задишка
