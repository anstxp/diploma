# Атерогенна дисліпідемія
icon: 💗
severity: medium
---
У вас комбінація **низький HDL ({{ hdl }} мг/дл) + високі тригліцериди ({{ trigly }} мг/дл)**. Це класична картина **атерогенної дисліпідемії** — типу порушень ліпідного обміну, який сильно підвищує серцево-судинний ризик.

{% if on_statins %}
Ви приймаєте статини, але ліпідограма все ще атерогенна — це означає що **поточної терапії недостатньо**. Не панікуйте: статини добре працюють на LDL, але мало впливають на HDL і тригліцериди. Потрібен перегляд схеми лікування з кардіологом.
{% elif has_diabetes %}
У вас цукровий діабет, який сам по собі підвищує серцево-судинний ризик. У сукупності з цією дисліпідемією ризик інфаркту/інсульту вищий — потрібен агресивніший контроль ліпідів. За рекомендаціями ESC/EAS цільові значення LDL **залежать від категорії ризику**: при діабеті високого ризику зазвичай <70 мг/дл, при дуже високому ризику (ССЗ в анамнезі, ушкодження органів-мішеней, тривалий діабет) — <55 мг/дл. Точну ціль визначить лікар на основі повної оцінки ризику.
{% elif has_cardiovascular_disease %}
Оскільки у вас вже була серцево-судинна подія (інфаркт, інсульт або діагностована ішемічна хвороба), такий ліпідний профіль **значно підвищує ризик повторної події**. Потрібна вторинна профілактика з агресивними цільовими значеннями.
{% else %}
Часто супроводжується інсулінорезистентністю, метаболічним синдромом, абдомінальним ожирінням. Хороша новина — навіть невеликі зміни способу життя дають значне покращення.
{% endif %}

actions:
{% if on_statins %}
- **Зверніться до кардіолога** — обговоріть посилення терапії (вища доза статинів, додавання езетимібу, фібратів або омега-3)
- Перевірте комплаєнс: чи приймаєте статини регулярно, у потрібний час, разом з їжею
- Здайте АЛТ і креатинкіназу — переконайтеся що поточна доза не дає побічних ефектів
- Перегляньте дієту і фізичну активність — навіть на статинах вони критично важливі
{% elif has_diabetes %}
- **Зверніться до ендокринолога або кардіолога** — обговоріть призначення статинів (показано усім діабетикам ≥40 років)
- LDL target залежить від категорії ризику: **<70 мг/дл при високому**, **<55 при дуже високому** (попередня СС-подія, ушкодження органів). Точну ціль обговоріть з лікарем
- Перегляд глікемічного контролю — гіперглікемія погіршує дисліпідемію
- Дієта: мінімум простих вуглеводів, алкоголю, трансжирів
{% elif has_cardiovascular_disease %}
- **Терміново до кардіолога** для перегляду вторинної профілактики
- Якщо ще не на статинах — мають бути призначені (цільовий LDL <55 мг/дл після події)
- Можуть знадобитися додаткові препарати: антитромбоцитарні, бета-блокатори, ACE-інгібітори
{% else %}
- Зверніться до сімейного лікаря або кардіолога для оцінки повного серцево-судинного ризику
- Здайте: HbA1c, повторно ліпіди натще, АЛТ (для оцінки печінки)
- Спосіб життя: середземноморська дієта, обмеження простих вуглеводів і алкоголю
- 150 хв/тиждень аеробної активності + 2 силові тренування
- Якщо є зайва вага — мета: 5-10% зниження маси
- При наявності інших факторів ризику — обговоріть статини з лікарем
{% endif %}
