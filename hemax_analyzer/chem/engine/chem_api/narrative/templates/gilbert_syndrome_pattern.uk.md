# Можливий синдром Жільбера
icon: 🧬
severity: low
---
Ваш загальний білірубін ({{ bilirubin_total }} мг/дл) підвищений, але ферменти печінки (АЛТ, АСТ, ЛФ) у нормі. Така **ізольована непряма гіпербілірубінемія** характерна для **синдрому Жільбера** — поширеної (3-7% населення) **спадкової особливості**, а не хвороби.

При синдромі Жільбера печінка трохи менш ефективно виводить непрямий білірубін, особливо під час стресу, голоду, інфекції. Це **не шкодить здоров'ю**, не потребує лікування і не впливає на тривалість життя.
---
actions:
- Зверніться до сімейного лікаря для виключення інших причин (вірусні гепатити, гемоліз)
- Здайте: фракціонований білірубін (прямий + непрямий), HBsAg, anti-HCV, ретикулоцити
- Якщо діагноз підтверджується — особливих дій не потрібно
- Уникайте тривалого голоду, надмірних навантажень — вони погіршують стан
- Деякі ліки гірше переносяться при синдромі Жільбера — обговоріть це з лікарем
---
red_flags:
- Жовтушність + біль у правому підребер'ї або тверда печінка — потрібне дообстеження
- Темна сеча або знебарвлений кал
- Жовтушність + слабкість, що погіршується
